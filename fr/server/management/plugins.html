Installer des plugins¶

Qu’est ce qu’un plugin ?¶

Les plugins sont des fichiers codés en Java qui changent la manière dont un serveur fonctionne, généralement en y ajoutant des fonctionnalités. Ils permettent aux administrateurs de faire beaucoup de modifications, comme ajouter un système d’économie, gérer les téléportations et les permissions, etc.

Trouver des plugins¶

Avertissement

Ne téléchargez pas des plugins à partir de sources non fiables ! Certains plugins malveillants peuvent être utilisés pour donner à des personnes malveillantes l’accès à votre serveur ou à votre ordinateur.

SpongePowered utilise actuellement la plateforme Ore pour faciliter la distribution et le téléchargement de plugins aux développeurs et utilisateurs de plugins. Alternativement, vous pouvez rechercher des plugins sur les forums de SpongePowerred.

Installation¶

SpongeForge¶

Afin d’installer des plugins, placez-les dans le dossier /mods/ de votre répertoire de jeu ou de seveur. Si votre téléchargement est venu dans un fichier .zip, alors vous devrez l’extraire afin de trouver un fichier .jar à l’intérieur.

Vous pouvez également placer vos plugins à l’intérieur du sous-dossier /mods/plugins/ ou même définir un dossier pesonnalisé dans le fichier global.conf via le paramètre plugins-dir. SpongeForge va automatiquement chercher des plugins à l’intérieur de ces dossiers. Notez que les plugins qui utilisent les Mixins doivent se trouver à l’intérieur du dossier mods.

SpongeVanilla¶

Note

Par souci de cohérence entre SpongeForge et SpongeVanilla, les plugins sont stockés dans le dossier mods sur SpongeVanilla.

Afin d’installer des plugins, placez-les dans le dossier /mods/ de votre répertoire de jeu ou de seveur. Si votre téléchargement est venu dans un fichier .zip, alors vous devrez l’extraire afin de trouver un fichier .jar à l’intérieur.

Vous pouvez également placer vos plugins à l’intérieur du sous-dossier /mods/plugins/ ou même définir un dossier pesonnalisé dans le fichier global.conf via le paramètre plugins-dir. SpongeVanilla va automatiquement chercher des plugins à l’intérieur de ces dossiers. Notez que les plugins qui utilisent les Mixins doivent se trouver à l’intérieur du dossier mods.

Problèmes courants¶

Si vous rencontrez des problèmes avec un plugin, prenez en compte les choses suivantes:

  • Le plugin est-il compatible avec votre version de Minecraft ? Alors que Sponge essaie de faire fonctionner d’anciens plugins, ce n’est parfois pas possible. Dans la plupart des cas, les plugins basés sur une version stable devraient continuer à fonctionner sans être mis à jour. Veuillez noter que les plugins peuvent cesser de fonctionner si vous passez à une autre version majeure de l’API, car les versions majeures contiennent des changements non rétrocompatibles. Reportez-vous à notre schéma de versioning :doc:<../../contributing/versioning> pour plus d’informations.
  • Utilisez-vous une version récente de Java? Sponge est fait pour Java 8 et est connu pour avoiravoir un bug dans la version de Java JRE 1.8.0_2040, assurez-vous donc d’utiliser une version plus récente que celle-ci.
  • Le plugin peut être obsolète. Existe t-il une version plus récente de ce plugin ?
  • Le plugin a-t-il besoin d’une implémentation spécifique et/ou d’une certaine version de Sponge ? Certains plugins peuvent contourner l’API Sponge ou autrement reposent sur des détails qui changent entre les versions ou les plateformes. Vérifiez sur le site où vous l’avez téléchargé.
