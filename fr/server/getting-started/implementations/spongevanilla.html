Installer SpongeVanilla¶

SpongeVanilla est une implémentation vanilla de SpongeAPI en tant que serveur autonome.

Vue d’ensemble¶

SpongeVanilla est une implémentation de SpongeAPI créée dans le but de compléter le serveur Minecraft Vanilla. Cela veut dire qu’il s’agit d’un serveur indépendant, qui ne requiert ni Minecraft Forge ni Forge mod loader (FML). SpongeVanilla est développé en parallèle à l’implémentation Forge de Sponge, en tant qu’alternative pour les utilisateurs qui ne veulent pas de serveur Forge. A l’origine il s’agissait d’un projet indépendant nommé Granite, par les développeurs AzareusNation et VoltaSalt. L’équipe SpongeVanilla à officiellement rejoint l’équipe de développement Sponge en mars 2015.

Télécharger¶

Review our Système de version and grab your copy of SpongeVanilla here.

Installer SpongeVanilla¶

Note

Si vous utilisez (ou envisagez d’utiliser) un hébergeur pour votre serveur, ils peuvent avoir un panneau de contrôle qui peut installer Sponge pour vous.

Note

Lors de l’utilisation de l’installateur de Mojang, Mojang utilise leur propre version de Java et non celle installée sur votre système d’exploitation. L’installateur actuel vient avec Java 1.8.0_74. Notez que Sponge requiert au minimum Java 1.8.0_20 ou plus pour fonctionner correctement, mais il est recommandé d’utiliser la dernière version de Java 8.

SpongeVanilla ne fonctionne que comme un serveur dédié.

  1. Téléchargez le fichier SpongeVanilla.jar sur le site de Sponge.
  2. Exécutez-le via la ligne de commande: java -jar spongevanilla-whatever.jar
  3. Configurez Redirection de port pour vous assurer que d’autres joueurs peuvent se connecter.

Avertissement

Ne double-cliquez pas sur le fichier .jar, cela lancera le serveur sans fenêtre pour le contrôler !
