Installer Git¶

Git est un logiciel de versionning libre de droit, il aide énormément pour le développement collaboratif.

Le projet Sponge, de même que des centaines d’autres projets libres de droits, est hébergé sur GitHub. Donc, Git est un outil crucial pour le développement de Sponge et de SpongeAPI.

Le site de Git dispose d’une documentation considérable, et leur page de téléchargement propose une large gamme d’options pour utiliser un interface graphique sur différents OS.

Télécharger¶

Windows¶

GitHub Desktop est un moyen facile d’installer Git sur Windows, car il est inclus comme une partie du logiciel.

Redémarrer votre ordinateur après avoir installé Git est recommandé.

Mac¶

Il a plusieurs façons d’installer Git sur macOS.

La méthode la plus simple est d’installer Xcode Command Line Tools.

Avertissement

Ces instructions ne fonctionnent pas sur les Macs utilisant une version plus ancienne que Mavericks. Si vous utilisez une version plus ancienne que Mavericks, installez le client GitHub à la place.

  1. Lancez le terminal.
  2. Exécutez xcode-select --install.
  3. Installez le, et mangez un cookie en attendant.
  4. Exécutez git depuis la Console.

Vous pouvez aussi installer GitHub Desktop. Git est disponible dans le cadre de l’installation de GitHub Desktop.

Redémarrer votre ordinateur après avoir installé Git est recommandé.

Linux and Unix¶

La méthode la plus simple d’installer Git sous Linux est d’utiliser le gestionnaire de package fourni avec votre distribution de Linux.

Note

Vous aurez peut-être besoin de précéder ces commandes avec sudo.

  1. Lancez le terminal.
  2. Si vous êtes sur Ubuntu ou une distribution basée sur Debian, exécutez apt-get install git. Si vous êtes sur Fedora, exécutez yum install git.

GitHub Desktop n’est actuellement pas disponible pour Linux, contrairement à Windows et Mac.

Redémarrer votre ordinateur après avoir installé Git est recommandé.

Réglages¶

Qui êtes vous?¶

Avant de commencer à travailler avec git et le repository, assurez-vous que la configuration de git a votre identité de configurée. Ouvrez votre CLI et entrez:

git config --list

Regardez votre user.name et user.email. Si ce ne sont pas les mêmes que votre nom d’utilisateur et email de votre compte GitHub, configurez les:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Astuce

Ne procédez à aucune contribution dans l’un des dépôt Sponge avant d’avoir établi votre user.name et votre user.email.
