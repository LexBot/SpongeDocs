Installer le JDK¶

Le Kit de Développement Java - également connu sous le nom le JDK - est nécessaire pour développer des plugins Sponge ou pour travailler sur Sponge lui-même.

Télécharger¶

Avant d’installer le JDK, désinstallez les anciennes versions de Java qui sont présentes sur votre ordinateur.

Note

Les contributeurs de Sponge et les développeurs de plugins doivent utiliser JDK 8, les anciennes versions de Java n’étant plus supportées. Il faut savoir que beaucoup de serveurs Minecraft ne sont toujours pas sur Java 8. Afin de pouvoir faire fonctionner Sponge et tous les plugins proprement, vous devrez passer sur Java 8 puisque Sponge ne peut pas fonctionner sur des versions antérieures (comme Java 6 et 7).

Oracle fournit des téléchargements gratuits du Kit de Développement Java sur leur site Web. Assurez-vous que vous installez le JDK (Java Development Kit), pas le JRE (Java Runtime Environment). Il y a une différence entre les deux.

  • Java Development Kit 8

Comme alternative open source vous pouvez également télécharger le JDK depuis AdoptOpenJDK.

Une fois le processus d’installation terminé, redémarrez votre ordinateur. Le JDK devrait alors être prêt à l’emploi.
