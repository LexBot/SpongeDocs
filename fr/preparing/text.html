Installer un Éditeur de Texte¶

Articles on SpongeDocs are saved as text files in the reStructuredText markup language. Although your operating system’s default text editor is likely sufficient for editing these files, installing a different text editor may prove to be useful.

Téléchargements¶

Visual Studio Code¶

VS Code est un éditeur de texte et de code open source et cross platform créé par Microsoft et sous la licence MIT. C’est un choix populaire utilisé par beaucoup de développeurs Sponge, pour de la documentation et même pour développer des plugins. Contrairement à un éditeur de texte classique, il supporte le debugging, la coloration de syntaxe, la complétion de code intelligente, la réfactorisation de code, et plus. Il y a également un support pour Git, un bon plugin pour le support de reST <https://marketplace.visualstudio.com/items?itemName=lextudio.restructuredtext>`_, et même un plugin pour HOCON.

VS Code peut être téléchargé gratuitement sur https://code.visualstudio.com/.

Atom¶

Atom est un éditeur de texte ouvert et publié sous la licence MIT. Atom possède un support intégré pour les projets stockés dans un dépôt Git, tels que SpongeDocs. Cet article va plus en profondeur dans les fonctionnalités de l’intégration de Git dans Atom. Si vous êtes intéressés dans l’ajout du support de reST dans Atom, regardez ce paquet.

Atom peut être téléchargé gratuitement depuis https://atom.io/.

Notepad++¶

Notepad++ est un des éditeur de texte les plus populaires.

Notepad++ peut être téléchargé gratuitement depuis https://notepad-plus-plus.org/.

Sublime Text¶

Sublime Text est un éditeur de texte multi-plateforme, avec quelques fonctions intéressantes. Il permet de personnaliser l’apparence de l’éditeur, de créer des macros utiles, et même d’inclure une API plugin pour rendre vos éditions encore plus simples. Sublime Text est disponible en version gratuite, et licence payante.

Sublime Text peut être téléchargé gratuitement depuis https://www.sublimetext.com/.
