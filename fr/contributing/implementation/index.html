Développer Sponge¶

Cette section décrit la meilleure façon d’aborder le projet lorsque vous avez l’intention de développer Sponge lui-même. Il est obligatoire de se familiariser avec git et notre Style de Code. Des connaissances basiques de la La structure du projet Sponge est également très utile.

Si vous voulez nous rejoindre dans la création de Sponge, dirigez-vous tout droit vers la page Soumettre une Contribution.
