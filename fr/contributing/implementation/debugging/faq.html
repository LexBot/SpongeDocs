IntelliJ¶

  • Comment réparer les configurations d’exécution où la classe principale n’est pas indiquée ?

    Quand vous êtes dans la fenêtre Edit Configurations..., sélectionnez le module qui contient .main dans son nom dans la liste déroulante Use classpath of module: . Cliquez sur Apply.

  • Pourquoi mon espace de travail SpongeVanilla n’a pas de tâche genIntelliJRuns ?

    C’est normal. Consultez le post VanillaGradle doesn’t have genIntelliJRuns sur nos forums pour plus d’informations.
