Préparation d’Eclipse¶

Cet article décrit comment configurer votre espace de travail Eclipse pour le développement de plugins avec l’API Sponge et un build system comme Maven et Gradle.

Note

Un tutoriel de Mumfrey montrant l’installation d’un espace de travail Sponge dans Eclipse, en utilisant les nouvelles fonctionnalités dans Buildship et ForgeGradle, peut être trouvé ici.

Gradle¶

Préparer Gradle¶

Note

Gradle est automatiquement inclus dans Eclipse à partir de la sortie de Neon (22 Juin 2016). Ces étapes ne sont requises que pour les version précédentes.

Vous devez d’abord installer Gradle Integration Plugin avant d’utiliser Gradle dans Eclipse. Cela ne doit être fait qu’à la création de votre premier projet.

Astuce

En général, vous n’avez pas besoin du module optionnel Spring, distribué avec ce plugin. Vous pouvez donc le décocher pendant l’installation.

  • Ouvrez Eclipse.
  • Cliquez sur Help > Eclipse Marketplace.
  • Cherchez Gradle Integration Plugin.
  • Installez le plugin Gradle Integration Plugin.

Créer votre projet¶

  • Ouvrez Eclipse.
  • Cliquez sur File > New > Other.
  • Sélectionnez Gradle > Gradle Project.
  • Cliquez sur Next.
  • Choisissez un nom de projet, puis cliquez sur Finish.

Éditer le script de build¶

  • Ouvrez build.gradle dans le navigateur.
  • Éditez le script de build conformément aux instructions à Configurer Gradle.
  • Cliquez-droit sur votre projet, et sélectionnez Gradle > Refresh Gradle Project.

Importer votre projet¶

Si vous avez déjà commencé votre projet et que vous souhaitez l’importer à nouveau à un stade ultérieur, vous devrez l’importer au lieu de le recréer à l’intérieur de votre IDE :

  • Cliquez sur File > Import
  • Sélectionnez Gradle > Gradle Project
  • Naviguez jusqu’au dossier racine du projet
  • Cliquez sur Finish

Maven¶

Créer votre projet¶

  • Ouvrez Eclipse.
  • Cliquez sur File > New > Other.
  • Selectionnez Maven sur le côté gauche du menu contextuel, puis sélectionnez Maven Project sur le menu de droite.
  • Cliquez sur Next.
  • Sélectionnez Create a simple project, tant que vous n’avez pas besoin d’une configuration plus avancée.
  • Entrez votre Group ID, Artifact ID, Version, votre nom de projet et votre description du projet.
    • Votre Group ID correspond généralement à votre nom de package Java. Voir Classe Principale du Plugin pour plus de détails.
    • Votre Artifact ID correspond généralement au plugin ID que vous avez choisi plus tôt, par exemple myplugin.
    • C’est à vous de choisir votre Version. Voir Numéros de Version pour plus de détails.
  • Cliquez sur Finish.

Astuce

Souvent Eclipse n’ouvre pas nouveaux projets après leur création. Dans ce cas, essayez de fermer l’écran de bienvenue d’Eclipse. Votre projet devrait être ouvert derrière celle-ci.

Éditer la configuration du projet¶

  • Ouvrez pom.xml dans le navigateur.
  • Éditez la configuration de build conformément aux instructions à Configurer Gradle.
  • Actualisez votre projet Maven.

Importer votre projet¶

Si vous avez déjà commencé votre projet et que vous souhaitez l’importer à nouveau à un stade ultérieur, vous devrez l’importer au lieu de le recréer à l’intérieur de votre IDE :

  • Cliquez sur File > Import
  • Sélectionnez Maven > Existing Maven Projects
  • Naviguez jusqu’au dossier racine du projet
  • Cliquez sur Finish
