Установка Git¶

Git — система управления версиями с открытым исходным кодом, которая очень сильно помогает в разработке проектов многими людьми.

The Sponge project, alongside thousands of other open-source projects, hosts its Git repositories on GitHub. Thus, Git is a crucial tool in the development of Sponge and SpongeAPI.

На сайте Git есть документация и на странице с загрузками есть различные клиенты с пользовательским интерфейсом.

Скачать¶

Windows¶

GitHub Desktop — самый простой метод установки Git для Windows, т. к. Git идёт в комплекте с программой.

Rebooting your computer after installing Git is recommended.

Mac¶

Есть несколько способов установки Git на macOS.

Самый простой способ — установка Xcode Command Line Tools.

Предупреждение

Эти инструкции не работают на компьютерах Mac под управлением версии старше Mavericks. Если вы используете версию OS X старше Mavericks, установите клиент GitHub.

  1. Запустите Терминал.
  2. Выполните xcode-select --install.
  3. Установите его. Пока вы ждёте, можете съесть несколько печенек.
  4. Выполните git в терминале.

Альтернативный вариант — установить GitHub Desktop. Git идёт в комплекте с программой.

Rebooting your computer after installing Git is recommended.

Linux и Unix¶

Самый простой способ установки Git на Linux — с помощью диспетчера пакетов, который уже встроен в дистрибутив Linux.

Примечание

Для выполнения команды, Вам может потребоваться префикс sudo.

  1. Запустите Терминал.
  2. Run apt-get install git if you are on an Ubuntu or Debian-based distribution. Run yum install git if you are on Fedora.

GitHub Desktop на данный момент недоступен для Linux, в отличие от Windows и Mac.

Rebooting your computer after installing Git is recommended.

Установка¶

Кто Вы такой?¶

Прежде чем Вы начнете работу с git и репозиторием, убедитесь, что ваша конфигурация git авторизованна. Откройте CLI и введите:

git config --list

Ищите user.name и user.email. Если их значения не совпадают с именем пользователя и адресом электронной почты как в GitHub, установите их:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Совет

Не проводите никаких правок в любом из репозиториев Sponge, до тех пор, пока вы не укажете правильно значения „“user.name““ и „“user.email““.
