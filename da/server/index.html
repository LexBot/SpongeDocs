Opsætning af en server¶

The articles in this section explain how to create and run a server powered by an implementation of SpongeAPI.

Hvis du er en bruger, der ønsker at oprette en ny server ved hjælp af en implementering af Sponge eller migrere en eksisterende server til en implementering af Sponge, er det for dig. Pluginudviklere kan også være interesserede i artiklerne i dette afsnit - testservere kan være nyttige til pluginudvikling.

Bemærk

We attempt to be implementation-agnostic when possible, meaning that instructions generally aren’t specific to one implementation of SpongeAPI. Nonetheless, mistakes can happen.
