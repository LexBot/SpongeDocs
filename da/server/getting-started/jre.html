Indstallere Java¶

Java er nødvendig for at køre Sponge og Minecraft: Java Editions. Du har sandsynligvis allerede Java, men du kan være nødvendigt at opdatere den.

Sponge requires Java 8 (specifically 1.8.0_20 or above) at this time. Older Java versions are deprecated and will not work with Sponge. The difference between major versions of Java (6, 7, 8) is significant, and older versions cannot run Sponge properly. Currently, Sponge will not run with versions of Java greater than 8 either.

Indstallere Java¶

Hvis du har Windows eller mac-OS til din computer, kan du › hente Java fra den officielle hjemmeside <https://java.com/en/download/manual.jsp>» __.

Linux-brugere kan installere OpenJDK via deres Pakkestyring. OpenJDK er open source-version af Oracle version af Java, og det bør arbejde lige så godt, hvis ikke bedre. Men det er også muligt at › hente Oracle versionen til Linux <https://www.oracle.com/technetwork/java/javase/downloads/index.html>‹ __, men vær opmærksom på at mange Java-afhængige Linux pakker stadig vil installere OpenJDK alligevel.

Du skal muligvis konfigurere stien til JRE/JDK i din

  • Opgavestarter ved hjælp af ›‹ Profile‹‹ (›‹ avancerede indstillinger ›‹)-> ›‹<Your Profile>‹‹-> ›‹ Java-eksekverbare ›‹ (›‹…/java.exe‹‹)
  • Server ved hjælp af variablen ›‹ PATH‹‹ eller den fulde sti til din eksekverbare java

32-Bit mod 64-bit¶

If your computer supports it, you should use 64-bit versions of Java whenever possible. The Java installers from the this website should detect whether your computer is ready for 64-bit. However, the autodetection is sometimes wrong (try different browsers, if you are unsure).

Det er også muligt at se dette op i systeminformationsvinduet.

  • Windows: Tryk på `` <Win> `` + `` <Pause> `` eller via `` Systeminformation`` kig på din `` systemtype``.
  • Linux: Åbn en terminal og skriv `` arch`` eller `` uname -m``. `` i686`` -> 32-bit. `` x86_64`` -> 64-bit.
  • Mac: Der er det alligevel kun en download …

Hvis du er sikker på, om dit systems arkitektur, kan du manuelt vælge <https://java.com/da/download/manual.jsp> `__ det rigtige installationsprogram selv.

Fordi 64-bit version af Java kører betydeligt bedre, og også lar Java bruge mere end ~ 3 GB RAM (hukommelse), anbefaler vi det altid over 32-bit.

De fleste moderne computere understøtter 64-bit.

JDK Mod. JRE¶

JRE (Java Runtime Environment) bruges til at køre Java-applikationer. Den overførte download-side giver JRE.

JDK (Java Development Kit) bruges til at oprette Java-programmer, og du behøver det ikke, medmindre du har planer om at lave Sponge plugins eller arbejde på Sponge. I nogle tilfælde kan du muligvis bruge JDK til at diagnosticere et kørende Java-program som Sponge. Du kan downloade JDK’en fra et andet websted <https://www.oracle.com/technetwork/java/javase/downloads/index.html> `__.
