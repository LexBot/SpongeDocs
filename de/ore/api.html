Ore Web API¶

Ore bietet eine JSON RESTful API an um mit Mods und Plugins, die von Ore indiziert sind, zu interagieren. Folgend ist die Spezifikation der API.

Datentypen¶

Es gibt ein paar Integer-basierte enumartige Datentypen, die an verschiedenen Stellen der API verwendet werden. Die folgende Liste enthält alle diese Datentypen.

Kategorie IDs:¶

0 Administrative Tools
1 Chat Tools
2 Entwickler Tools
3 Wirtschaft
4 Spielmechaniken
5 Spiele
6 Schutz
7 Rollenspiel
8 Welt-Management
9 Verschiedenes
10 Undefiniert

Sortiermethoden IDs:¶

0 Meisten Sterne
1 Meisten Downloads
2 Meisten Ansichten
3 Neuste
4 Kürzlich aktualisiert

Veraltete Routen¶

The following routes are deprecated and should not be used. The replacements are likely to differ from the old one, so consult the docs before migrating.

Veraltet Ersatz
/api/projects /api/v1/projects
/api/users /api/v1/users
/api/projects/:pluginId/version /api/v1/projects/:pluginId/version
/api/projects/:pluginId/versions/:version /api/v1/projects/:pluginId/versions/:version
/api/projects/:pluginId /api/v1/projects/:pluginId
/api/users/:username /api/v1/users/:username
/api/projects/:pluginId/versions/:name /api/v1/projects/:pluginId/versions/:name
