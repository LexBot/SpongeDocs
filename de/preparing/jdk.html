Installieren des JDKs¶

Das Java Development Kit (JDK) wird benötigt um Sponge oder Sponge-Plugins zu entwickeln.

Download¶

Vor der Installation des JDKs, sollten alle älteren Versionen von Java von dem Computer entfernt werden.

Bemerkung

Sponge-Contributors und Plugin-Autoren müssen JDK 8 benutzen, weil ältere Versionen auf JAva nicht länger unterstützt werden. Beache, dass einige Minecraft-Server immer noch nicht zu Java 8 umgestiegen sind. Um Sponge und seine Plugins richtig nutzen zu können, musst du zu Java 8 aktualisieren, weil Sponge auf älteren Versionen (z.B. Java 6 und 7) nicht laufen wird.

Oracle bietet den kostenlosen Download des Java Development Kit auf ihrer Website an. Stellen Sie sicher, dass Sie das Java Development Kit (JDK) und nicht das Java Runtime Environment (JRE) installieren. Es gibt einen Unterschied zwischen den beiden.

  • Java Development Kit 8

Als eine OpenSource-Alternative kannst du dir auch die JDK von AdoptOpenJDK herunterladen.

Wenn die Installation fertig ist, starte deinen Computer neu. Das JDK sollte jetzt nutzbar sein.
