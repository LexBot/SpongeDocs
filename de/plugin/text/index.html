Text¶

Formatierter Text kann mit Hilfe eines Text.Builders erstellt werden. Die Text-API kann auf vielen Wegen zum Kombinieren von Styles, Farben und Textaktionen (TextAction) verwendet werden.

Die Pagination ist der Prozess mit dem Inhalte (wie eine Liste an Texten) in einzelne Seiten unterteilt werden. Dieser Vorgang wird auch in diesem Abschnitt diskutiert.
