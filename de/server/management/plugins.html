Installieren von Plugins¶

Was sind Plugins?¶

Plugins sind Dateien, geschrieben in Java, die die Art wie der Server funktioniert verändern. Dies geschieht hauptsächlich um den Server um Funktionen zu erweitern. Sie erlauben Server-Besitzern Modifikationen vorzunehmen, beispielsweise durch Hinzufügen eines Wirtschaftssystems, Managen von Teleportern und Berechtigungen, etc.

Finden von Plugins¶

Warnung

Lade Plugins nur von vertrauenswürdigen Quellen herunter! Schädliche Plugins können verwendet werden, um Fremden unerlaubten Zugriff auf deinen Server oder Computer zu ermöglichen.

SpongePowered betreibt aktuell die Ore-Plattform um es für Plugin-Developer und Benutzer einfach zu machen Plugins zu verteilen und herunterzuladen. Alternativ, kannst du auch für Plugins in den SpongePowered-Forums suchen.

Installation¶

SpongeForge¶

Um Plugins zu installieren, musst du diese im /mods/ Ordner deines Spieles oder deines Server platzieren. Wenn dein Download in Form einer .zip-Datei kam, musst du diese gegebenenfalls entpacken, um an die enthaltene .jar-Datei zu kommen.

Du kannst deine Plugins auch im /mods/plugins/ Unterordner oder sogar einem beliebigen Order platzieren, wenn du ihn in der global.conf Datei über die plugins-dir Option eingetragen hast. SpongeForge wird diese Ordner automatisch nach Plugins durchsuchen. Bitte beachte, dass Plugins, die Mixins verwenden, im mods Ordner liegen müssen.

SpongeVanilla¶

Bemerkung

Der Einheitlichkeit halber zwischen dem SpongeForge und SpongeVanilla werden Plugins im „mods“-Ordner hinterlegt, wenn du einen SpongeVanilla-Server betreibst.

Um Plugins zu installieren, musst du diese im /mods/ Ordner deines Spieles oder deines Server platzieren. Wenn dein Download in Form einer .zip-Datei kam, musst du diese gegebenenfalls entpacken, um an die enthaltene .jar-Datei zu kommen.

Du kannst deine Plugins auch im /mods/plugins/ Unterordner oder sogar einem beliebigen Order platzieren, wenn du ihn in der global.conf Datei über die plugins-dir Option eingetragen hast. SpongeVanilla wird diese Ordner automatisch nach Plugins durchsuchen. Bitte beachte, dass Plugins, die Mixins verwenden, im mods Ordner liegen müssen.

Häufige Probleme¶

Wenn du Probleme mit einem Plugin hast, prüfe folgendes:

  • Ist das Plugin kompatibel mit deiner Minecraft Version? Sponge versucht alte Plugins weiterhin laufen zu lassen, allerdings ist das nicht immer möglich. In den meisten Fällen sollten Plugins, die auf Stable-Releases basieren, weiter funktionieren, ohne das Updates benötigt werden. Bitte beachte, dass Plugins möglicherweise aufhören zu funktionieren, wenn du auf eine andere Haupt-Version der API wechselst, da andere Haupt-Versionen inkompatible Änderungen enthalten können. Für mehr Informationen lies dir unser Versionungs-Schema durch.
  • Nutzt du die aktuelle Java Version? Sponge wurde für Java 8 entwickelt und ist bekannt dafür, einen Bug in JRE Versionen älter als 1.8.0_20 auszulösen. Stelle also sicher das deine Java Version neuer ist.
  • Eventuell ist das Plugin veraltet. Existiert eine neuere Version des Plugins?
  • Braucht das Plugin eine bestimmte Implementierung und/oder Build von Sponge? Manche Plugins können die Sponge-API umgehen oder anderweitig auf Details zugreifen, die zwischen Versionen und Plattformen verschieden sind. Prüfe dies auf der Seite, wo du das Plugin heruntergeladen hast.
