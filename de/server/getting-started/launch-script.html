Ein Start-Skript erstellen¶

Bemerkung

Diese Anleitung wird nur benötigt, wenn du einen Minecraft-Server auf deinem eigenen PC betrieben möchtest. Die meisten Minecraft-Server-Anbieter erstellen das Start-Skript automatisch.

Ein Start-Skript schreiben¶

Öffne zuerst einen Texteditor, wie zum Beispiel Atom, Sublime Text oder Notepad, und füge dann ein Start-Skript ein. Einige Beispiele für Start-Skripte für Windows, macOS und Linux, beachte dabei bitte die RAM-Grenze deines Computers.

Bemerkung

Die folgenden Beispiele sind allgemein. Um einen Forge-Server, der den Sponge-Coremod verwendet, zu starten, ändere forge-1.12.2-XYZ-universal.jar in die entsprechende Version, die sich in deinem Serververzeichnis befindet. Um einen SpongeVanilla-Server zu starten ändere den Dateinamen forge-1.12.2-XYZ-universal.jar in den Namen der SpongeVanilla.jar Datei.

Windows¶

java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar
pause

Speicher dein Windows-Start-Skript als launch.bat.

Warnung

Launching your server in a Command Prompt or PowerShell environment can lead to the server hanging due to the Quick Edit Mode of these shells. This mode freezes the process when you highlight something or click inside the console window. During this time, messages captured in the log will state that the server skipped many seconds or minutes worth of ticks. You can prevent this from occurring by not highlighting any text on the screen and not clicking inside the window, or by disabling the Quick Edit Mode in the Properties dialogue.

macOS¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Speicher dein Mac OS X-Start-Skript als launch.command.

Linux¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Speicher dein Linux-Start-Skript als launch.sh.

Ein Start-Skript ausführen¶

Stelle sicher, dass du dein Start-Skript aus einem Ordner, den du für den Server angelegt hast, ausführst. Dies ist nur für dich; leider kann Spongie deine Tränen nicht trocknen, wenn du diesen Rat nicht befolgst.

Du kannst das Start-Skript durch einen Doppelklick starten. Falls du eine Konsole oder ein Terminal einsetzt, navigiere zum Serverordner und führe es dort aus. Beachte, dass du der Mojang-EULA zustimmen musst, um einen Server ausführen zu können.

Bemerkung

Die Standard Minecraft Server GUI-Konsole ist durch Sponge deaktiviert, da sie sehr prozessorintensiv ist.

Warnung

Wenn du bei dem Versuch deinen Server auf OS X zu starten einen Berechtigungsfehler erhälst, versuche Folgendes:

  • Öffne das Terminal.
  • Gib chmod a+x mit einem Leerzeichen am Ende ein.
  • Zieh dein Start-Skript in das Terminal.
  • Drücke die Eingabetaste.
