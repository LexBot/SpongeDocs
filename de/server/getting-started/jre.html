Java installieren¶

Java wird zum Ausführen von Sponge und Minecraft: Java Edition benötigt. Du hast Java wahrscheinlich bereits installiert, aber musst es möglicherweise aktualisieren.

Sponge benötigt Java 8 (genauer: 1.8.0_20 oder neuer). Ältere Java-Versionen sind veraltet und funktionieren nicht mit Sponge. Die Unterschiede zwischen den Hauptversionen von Java (6, 7, 8) sind ziemlich groß und ältere Versionen können Sponge nicht korrekt ausführen. Zurzeit läuft Sponge nur mit Java 8.

Java installieren¶

Wenn du Windows oder MacOS auf deinem Computer nutzt, kannst du Java von der offiziellen Seite herunterladen.

Linux-Anwender können OpenJDK mithilfe ihres Package-Managers installieren. OpenJDK ist die Open-Source-Version von Oracles Java-Version und sollte genauso gut oder besser funktioniert. Es ist allerdings auch möglich Oracles Java Version herunterzuladen, manche Linux-Packages werden allerdings dennoch OpenJDK installieren.

Vielleicht musst du den Pfad zum JRE/JDK konfigurieren

  • der Launcher benutzt Profil (Erweiterte Optitionen) -> <Mein Profil> -> Java-Datei (.../java.exe)
  • Server benutzt die PATH Variable oder benutze den vollen Pfad um Java auszuführen

32-Bit vs. 64-Bit¶

Falls dein Computer eine 64-Bit-Version von Java unterstützt, solltest du diese aufjeden Fall benutzen. Der Java-Installer von dieser Website sollte erkennen, ob dein Computer 64-Bit unterstützt. Die automatische Erkennung ist manchmal aber falsch (versuche verschiedene Browser zu benutzen, falls du dir nicht sicher bist).

Es ist auch möglich, in das System-Info-Fenster zu sehen.

  • Windows: Drücke Super und Pause oder du suchst in Systeminfirmationen Sytemtyp.
  • Linux: Öffne ein Terminal und tippe arch or uname -m. i686 -> 32-bit. x86_64 -> 64-bit.
  • Mac: Es gibt nur einen Download…

Wenn Sie Ihre Systemarchitektur selbst wissen, können sie den passenden Installer auch ‚manuell wählen <https://java.com/en/download/manual.jsp>‘ __.

Da die 64-Bit-Version von Java viel besser läuft und außerdem ermöglicht, mehr als ~3 GB RAM (Arbeitsspeicher) zu verwenden, empfehlen wir sie anstelle der 32-Bit-Version.

Die meisten aktuellen Computer unterstützen 64-Bit.

JDK vs. JRE¶

Die JRE (Java Runtime Environment) ist nötig, um Java-Programme ausführen zu können. Die Downloadseite ist oben verlinkt.

Das JDK (Java Development Kit) wird verwendet um Java-Anwendungen zu schreiben und sollte nicht benötigt werden sollte man nicht vorhaben ein Sponge-Plugin zu erstellen oder an Sponge zu arbeiten. In manchen Fällen wird das JDK dennoch benötigt um laufende Java-Anwendungen, wie Sponge, zu diagnostizieren. Es ist möglich das JDK von einer anderen Seite herunterzuladen.
