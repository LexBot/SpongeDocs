SpongeForge installieren¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it’s more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Nutzer, die nicht Minecraft Forge verwenden möchten, sollten sich SpongeVanilla ansehen.

Download¶

Review our Versioning and grab your copy of SpongeForge here.

SpongeForge installieren¶

Bemerkung

Wenn du einen Spiel-Hoster verwendest (oder es planst), hast du vielleicht im Control Panel die Möglichkeit, Sponge zu installieren.

Bemerkung

Wenn du den Mojang-Installer benutzt, nutzt Mojang eine eigene Java-Version und nicht die, die auf deinem System installiert ist. Der Installer nutzt aktuell Java 1.8.0_74. Beache, dass Sponge mindestens 1.8.0_20 oder höher braucht, um korrekt zu funktionieren, es wird empfohlen die letzte Java 8-Version zu benutzen.

Einzelspieler / In-Game-LAN-Server¶

  1. Lade den Minecraft Forge-Installer von der Minecraft Forge-Webseite herunter. Stelle sicher, dass du exakt die gleiche Build Nummer verwendest wie oben angegeben.
  2. Starte den Forge Installer, welcher ein neues Profil in deinem Minecraft-Launcher erstellen wird.
  3. Öffne den Minecraft-Launcher und wähle das neue Forge-Profil aus.
  4. Klicke auf „Optionen“ und anschließend auf „Open Game Dir“.
  5. Lade dir SpongeForge von der Sponge Webseite herunter und verschiebe die Datei in den Ordner namens „mods“. Erstelle ihn, falls er nicht existiert.
  6. Sponge funktioniert im Singleplayer wie auch, wenn du deine Welt im LAN verfügbar machst.

Als Nächstes lerne wie man Sponge konfigurieren kann und wie man seinen Server verwaltet (beinhaltet das Installieren von Plugins).

Dedizierte Server¶

Bemerkung

Wenn du bereits einen Forge-Server hast, kannst du einfach SpongeForge in den mods-Ordner packen. Vergiss nicht, dass du Forge ggf. auf die Version aktualisieren musst, die SpongeForge benötigt. Wirf einen Blick auf den Anfang dieser Seite, wenn du dir nicht sicher bist, welche Version du benötigst.

Forge über die Kommandozeile installieren¶

  1. Besuche die Minecraft Forge Seite und klicke auf „Show all downloads“ um alle verfügbaren Versionen anzuzeigen. Suche dir die passende Version heraus, die im Dateinamen von SpongeForge angegeben ist und fahre über das (i) neben „Installer“ um den direkten Downloadlink zu bekommen.
  2. Nutze deine übliche Downloadmethode um die jar Datei herunterzuladen. Beispiel: wget http://url.to/forge-version-installer.jar
  3. Führe im Ordner in den du Forge installieren möchtest die jar Datei mit der --installServer Option aus. Beispiel: java -jar forge-version-installer.jar --installServer
  4. Fahre mit dem Hinzufügen von SpongeForge zu Forge fort.

Forge mit der GUI installieren¶

  1. Lade den Minecraft Forge-Installer von der Minecraft Forge-Webseite herunter. Wähle die Version, die im Dateinamen von SpongeForge genannt wird. Siehe oben für das Benennungsschema von SpongeForge und Forge.
  2. Führe den Forge-Installer aus und klicke auf „Install Server“, wähle einen leeren Ordner aus, in dem die Server-Dateien platziert werden sollen, und klicke anschließend auf OK.
  3. Fahre mit dem Hinzufügen von SpongeForge zu Forge fort.

SpongeForge zu Forge hinzufügen¶

  1. Lade die SpongeForge von der Sponge Homepage herunter und lege sie in den Ordner mods in deinem Serververzeichnis ab. Erstelle das Verzeichnis falls es noch nicht existiert.
  2. Jetzt kannst du den Server mit dem Startscript oder dem Startbefehl java -jar forge-version-XYZ.jar starten.
  3. Falls dein Server hinter einer Firewall steht, schau dir Portweiterleitung an, um sicherzustellen das sich andere Spieler verbinden können.

Als Nächstes kannst du ein Start-Skript erstellen, mehr über das Konfigurieren von Sponge und wie man den Server verwaltet (und wie man Plugins installiert) lernen.
