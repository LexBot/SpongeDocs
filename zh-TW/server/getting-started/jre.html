安裝 Java¶

執行 Sponge 和 Minecraft: Java Edition 都需要 Java。你應該已經安裝過了，但可能需要更新。

目前 Sponge 必須使用 Java 8（更明確的說法是 1.8.0_20 或更高版本）。舊版的 Java 已被棄用，無法用於 Sponge。Java 的主要版本（6、7、8）間的差異甚大，故較舊的版本無法正常執行 Sponge。目前 Sponge 亦無法以高於 Java 8 的版本執行。

安裝 Java¶

如果你的電腦使用 Windows 或 macOS，你可以 從官網下載 Java。

Linux 使用者可以透過他們的套件管理器來安裝 OpenJDK。OpenJDK 是開源版本的 Oracle Java，OpenJDK 應該會跟 Oracle Java 一樣正常運作。然而你依然可以 下載 Linux 專用的 Oracle 版本，但請留意，很多依賴 Java 的 Linux 套件依然會安裝 OpenJDK。

你可能需要自行設定你的 JRE/JDK 路徑

  • 啟動器使用 Profile（Advanced Options）-> <你的設定檔> -> Java-Executable（.../java.exe）
  • 伺服器使用 PATH 變數或使用 Java 執行檔的完整路徑

32 位元 vs. 64 位元¶

在您電腦支援的情況下你應該盡可能選擇使用 64 位元的 Java。這個網站上的 Java 安裝程式應該會檢測您的電腦是否準備好使用 64 位元。然而自動檢測有時也會出錯（如果您無法確定，請嘗試使用不同的瀏覽器）。

你也可以從系統資訊視窗中找到這個。

  • Windows：按下 <Win> + <Pause> 或從 系統資訊 查看您的 系統類型。
  • Linux：開啟一個終端機並輸入 arch 或是 uname -m。i686 -> 32 位元。x86_64 -> 64 位元。
  • Mac：反正也只有一種下載…

如果您很確定自己的系統類型，可以自行手動選擇正確的安裝程式。

由於 64 位元版的 Java 運作得更好，而且可以使用超過 3GB 的 RAM（記憶體），所以我們一直都推薦用它來取代 32 位元版。

現代的電腦大部分都能支援 64 位元。

JDK vs. JRE¶

JRE（Java Runtime Environment）用於執行 Java 應用程式。上方的下載頁面連結提供的是 JRE。

JDK（Java Development Kit）是用來建立 Java 應用程式的，除非你打算撰寫 Sponge 插件或是開發 Sponge，否則你不需要它。然而在某些狀況中，你可能需要 JDK 來診斷 Java 應用程式（例如 Sponge）。你可以在另一個網站中下載 JDK。
