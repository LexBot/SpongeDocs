安裝 SpongeVanilla¶

SpongeVanilla 是一個 SpongeAPI 的實作，以原版 Minecraft 伺服器為基礎。

概觀¶

SpongeVanilla 透過修改原版 Minecraft 伺服器來實作 SpongeAPI。這代表它是獨立的伺服器，不仰賴 Minecraft Forge，也不需要 Forge 模組載入器（FML）。SpongeVanilla 的開發與 Forge 版本的 Sponge 同時進行，作為一個不使用 Forge 的替代方案。SpongeVanilla 原本是一個名為 Granite 的獨立專案，由 AzureusNation 和 VoltaSalt 共同開發，已於 2015 年三月正式併入 Sponge 開發團隊。

下載¶

Review our 版本规范 and grab your copy of SpongeVanilla here.

安裝 SpongeVanilla¶

備註

如果你打算使用遊戲伺服器主機，主機商可能會有個控制面板，供你安裝 Sponge。

備註

当使用 Mojang 的安装程序时，Mojang 使用的是它们自己的 Java 版本而不是你的服务器提供的 Java 版本。安装程序已经内置了 1.8.0_74 版本的 Java。请注意 Sponge 需要不低于 1.8.0_20 的 Java 版本才可以使用；尽管如此，我们仍然推荐使用最新的 Java 8 构建。

SpongeVanilla 僅能以獨立伺服器形式運作。

  1. 從 Sponge 網站上下載 SpongeVanilla.jar。
  2. 透過命令列執行：java -jar spongevanilla-whatever.jar
  3. 設定連接埠轉送確保其他人也能連線。

警告

請不要點兩下 .jar 檔案，這樣會在沒有控制視窗的情況下啟動伺服器！
