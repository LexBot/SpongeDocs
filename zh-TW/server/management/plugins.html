安裝插件¶

插件是什麼¶

插件是使用 Java 撰寫來改變伺服器運作方式的檔案，通常是增加功能。它們允許伺服器擁有者做出許多修改，如添加經濟系統、管理傳送與權限等。

尋找插件¶

警告

不要從不可信任的來源下載插件！惡意插件可能會導致你的伺服器或者電腦被入侵

SpongePowered 目前運行 Ore 平台 以方便插件開發者與使用者發布與下載插件。此外，你也可以在 SpongePowered 論壇搜尋插件。

安裝¶

SpongeForge¶

為了安裝插件，請直接將它們放入你的遊戲或伺服器的 /mods/ 資料夾。如果你下載到 .zip 檔案，那你可能需要將其解壓縮並找到裡面的 .jar 檔案。

你也可以將你的插件放到 /mods/plugins/ 子資料夾，甚至透過修改 global.conf 檔案中的 plugins-dir 設定自訂資料夾。SpongeForge 將自動在這些資料夾搜尋插件。請注意，使用了 Mixins 的插件必須放在 mods 資料夾。

SpongeVanilla¶

備註

為了使 SpongeForge 與 SpongeVanilla 保持一致，SpongeVanilla 的插件也直接放在 mods 資料夾。

為了安裝插件，請直接將它們放入你的遊戲或伺服器的 /mods/ 資料夾。如果你下載到 .zip 檔案，那你可能需要將其解壓縮並找到裡面的 .jar 檔案。

常見問題¶

如果一個插件出現了問題，請考慮以下事項：

  • 插件是否與你的 Minecraft 版本相容？雖然 Sponge 試著保持舊版插件運作，但有時是不可能的。在大多數的情況，基於穩定版的插件應該無須更新仍可繼續運作。請注意，若你切換至不同主版號的 API，插件仍可能停止運作，因為主版號意味著包含不向下相容的破壞性更新。參閱我們的 版本規範 以取得更多資訊。
  • 你有運行正確的 Java 版本嗎？Sponge 是以 Java 8 建置，且目前已知在 JRE 1.8.0_20 中會觸發一個錯誤，所以請務必確認你使用的是更新的版本。
  • 插件可能已經過期，該插件使否有更新的版本？
  • 該插件是否需要特定的 Sponge 實作或建置？有些插件可能繞過 Sponge API，或者基於在版本或平台之間有所差異的細節。到你下載該插件的網站確認。
