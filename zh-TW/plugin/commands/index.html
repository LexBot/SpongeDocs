插件指令¶

小訣竅

由於指令 API 使用了文本 API，請務必先閱讀文本。

在 Sponge 中有兩種不同的 API 可建立指令： CommandCallable 介面和 CommandSpec 生成器（Builder）。

建立指令最方便的方式是使用 CommandSpec 生成器，這將會在本章節詳細介紹。它支援子指令與引數剖析。

另外，你可以使用 CommandCallable，一個能夠存取原始指令資料的低階介面。詳情請參閱這個頁面。
