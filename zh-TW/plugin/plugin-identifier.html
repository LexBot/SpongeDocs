插件識別碼¶

Sponge 插件包含一个唯一的 plugin ID 作为标识符，和一个可读的 plugin name 。为项目选取一个合适的名字十分重要，因为后续开发的插件会使用你之前选择的插件 ID（plugin ID）进行交互（例如插件之间的依赖）。插件 ID 同样用于创建默认的配置文件夹。

備註

插件 ID 的所有字母必须是小写，并且必须以字母开头。除字母外，插件 ID 只允许包含数字、短横线和下划线。不过，插件名称并没有这些限制，所以说在插件名称中包含空格等其他特殊字符也是完全可以接受的。插件 ID 总长度不得超过 64 个字符。

请牢记你的插件 ID 将是用于识别你的插件的主要标志，并会使用在其他插件的依赖、你的配置文件、以及存储和你的插件相关的其他属性。这也是为什么选用一个合适的插件 ID 十分重要，因为后来的修改会变得十分困难。

请前往 主要插件類別 页面，接着了解你的插件主类的设置方法。
