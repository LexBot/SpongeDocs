安裝 Git¶

Git 是一款開源的版本控制系統，對於協同專案開發提供了相當大的協助。

Sponge 專案與數以千計的開源專案一同將其 Git 儲存庫託管於 GitHub。因此，Git 是 Sponge 與 Sponge API 開發中的關鍵工具。

Git 網站 上有大量的文件，其下載頁面對於不同的作業系統提供了一系列的 GUI 客戶端選擇。

下載¶

Windows¶

GitHub Desktop 是一種在 Windows 安裝 Git 的簡易方式，因為 Git 已被作為軟體的一部份包含於其中。

建議在安裝 Git 之後重新啟動電腦。

Mac¶

在 macOS 安裝 Git 有數種方法。

最簡單的方法是安裝 Xcode Command Line Tools。

警告

這些說明不是用於運行比 Mavericks 更舊版本的 Mac，如果你運行的是比 Mavericks 更舊的 OS X 版本，請安裝 GitHub 客戶端。

  1. 啟動終端機
  2. 執行 xcode-select --install。
  3. 安裝它，並在等待時抓幾塊餅乾。
  4. 在終端機執行 git 。

此外，你可以安裝 GitHub Desktop。Git 可作為 GitHub Desktop 安裝中的一部份來取得。

建議在安裝 Git 之後重新啟動電腦。

Linux 與 Unix¶

在 Linux 安裝 Git 最簡單的方法是使用你的 Linux 發行版的套件管理系統。

備註

你可能需要在這些指令前加上 sudo。

  1. 啟動終端機
  2. 若使用 Ubuntu 或基於 Debian 的發行版，執行 apt-get install git。若使用 Fedora ，執行 yum install git。

不像 Windows 與 Mac，Linux 目前無法使用 GitHub Desktop。

建議在安裝 Git 之後重新啟動電腦。

配置¶

你是誰？¶

在開始使用 git 與儲存庫之前，確保你的 git 設定已包含了你的身分，開啟你的 CLI 並輸入：

git config --list

尋找 user.name 與 user.email。如果它們與你在 GutHub 帳號的使用者名稱與電子信箱不同，請設定它們：

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

小訣竅

在你建立你的 user.name 與 user.email 之前，請不要對 Sponge 儲存庫進行任何貢獻。
