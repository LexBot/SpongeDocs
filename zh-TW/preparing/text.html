安裝文字編輯器¶

SpongeDocs 上的文章使用名為 reStructuredText 的標記語言儲存於文字檔。雖然你的作業系統中預設的文字編輯器可能足以編輯這些檔案，但安裝一個不同的文字編輯器可能會很有用。

下載¶

Visual Studio Code¶

VS Code 是一個微軟基於 MIT 授權協議開源的跨平台程式碼與文字編輯器，它在 Sponge 開發者中是個熱門選擇，被用於撰寫文件甚至開發插件。與一般文字編輯器不同的是，它支援除錯、語法高亮、智慧型程式碼完成、程式碼重構等等。還有 Git 支援，對於 reST 支援的偉大插件，甚至是對於 HOCON 的插件。

VS Code 可在 https://code.visualstudio.com/ 免費下載

Atom¶

Atom 是一款基於 MIT 授權條款發布的開源文字編輯器，Atom 內建了對於儲存於 Git 儲存庫的專案（如 SpongeDocs）的支援。這篇文章 更深入介紹了 Atom 的 Git 整合功能。如果你有興趣為Atom 加上 reST 支援的話，請參考 這個套件.

Atom 可在 https://atom.io/ 免費下載。

Notepad++¶

Notepad++ 照理說是最受歡迎的文字編輯器之一。

Notepad++ 可在 https://notepad-plus-plus.org/ 免費下載。

Sublime Text¶

Sublime Text 是一個具備很多實用功能的跨平台文字編輯器。它允許使用者自定義編輯器外觀、建立實用的巨集，甚至包含了差件 API 來讓你更輕鬆地編輯。Sublime Text 同時提供了免費與付費版本。

Sublime Text 可在 https://www.sublimetext.com/ 免費下載。
