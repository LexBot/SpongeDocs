Sponge 文件¶

簡介¶

歡迎來到 SpongeDocs，開源專案 Sponge 的官方文件。

SpongeAPI is your all new plugin API for Minecraft: Java Edition. You can run it on a plain vanilla server (as SpongeVanilla) or on your fully modded MinecraftForge server (as SpongeForge). Check out our 版本控制方針 to understand selecting a file and grab your copy from our downloads page.

不確定想找什麼嗎？試著閱讀 關於 Sponge 專案 ，其中包含了一段 簡介 、我們的 常見問題 ，以及一篇關於 Sponge 專案的結構 的文章。

伺服器擁有者應閱讀我們在 建立伺服器 中的提醒和說明，包括如何安裝、執行並維護基於 Sponge 的伺服器。

Sponge 專案分成了幾個主要的部份。欲瞭解更多有關專案的結構，請到 這裡 或是前往 GitHub 儲存庫︰

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore | SpongeAuth

備註

Sponge development is ongoing. Stable and Experimental builds of both SpongeForge and SpongeVanilla are currently available for download.
