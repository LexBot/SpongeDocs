論壇發佈指南¶

在發佈前閱讀¶

如果您對 Sponge 感到困惑或好奇，我們強烈建議您閱讀常見問題。

請不要發佈要求提供 Sponge 的基本資訊的主題，如 「Sponge 是 Forge 模組嗎」、「Sponge 何時會發佈」 等。這些問題在 SpongeDocs 已有答案。

在向 Sponge 論壇發佈新主題之前，請確保問題或資訊沒有重複！可透過使用搜尋功能（頁面右上角），或是在建立新主題時選擇上方顯示的 Your topic is similar to... 框（當你在撰寫文章時會自動更新）。如果已有類似的主題，你便應該到該主題繼續討論！

允許討論其他 API，前提是不得口出惡言。

還需要更多資訊嗎？參考Sponge 文件中的 Sponge 資源、檢視論壇公告，或是跟隨我們的 Twitter，也可以緊盯 Sponge GitHub 來追蹤 Sponge 的進度。如果你真心想協助開發，請閱讀為 Sponge 貢獻 。

論壇規則¶

  • Sponge 論壇規則和指南的完整清單可以在 论坛、Discord 及 IRC 守则 中找到。

為您的文章分類¶

發表新文章時，使用文章標題右側的下拉式選單選擇其分類。試著確保這個類別符合你發佈的內容。 如果你不知道該類別的含義，在下拉式選單中的每一項旁邊都有說明。

感謝您的閱讀，歡迎來到 Sponge 社群！
