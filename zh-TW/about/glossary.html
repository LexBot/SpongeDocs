Sponge 術語¶

這是 Sponge 專案中常用的保留詞，短語和縮寫的指南。某些情況下會提供連結以提供更多信息。這並不能作為 SpongeAPI 或 Java 的指南。

備註

詞彙定義的來源是參考自 Minecraft 和/或 Sponge 專案。

定義¶

API
應用程式開發介面 - 一種使 modding 更輕鬆的方法。
貢獻者
那些為 SpongePowered 專案貢獻程式碼的人。
核心模組（Coremod）
核心模組是一種於載入 Mojang 類別（class）前事先載入的 Forge 模組，使它能透過 Access Transformers、改寫 ASM 類別或是用其他 Java 字節碼（bytecode）操作來修改原生的 Minecraft 程式碼。
Crowdin
一種基於 Web 的文件翻譯系統。SpongeDocs 利用它來提供多種語言的翻譯文件。https://crowdin.com/project/sponge-docs
開發人員
編寫程式碼的人（Sponge、插件、模組、……）。
Discord
Discord 是个免费的语音、视频及文字聊天软件。和 IRC 一样，Sponge 在 Discord 上有聊天室（即 Discord 服务器）。Sponge 的 Discord 服务器和 IRC 频道保持同步。
FLARD
它綠綠的、可食用且相當火爆。你不理他他就不會理你。
Forge
一個歷史悠久的 Minecraft modding API，有獨立安裝程式。Forge 能為單人遊戲、區網和專用伺服器啟用模組。 https://mcforge.readthedocs.io/en/latest/
GitHub
基於 Web、自由、免費且公開的原始碼分享以及專案托管系統。https://github.com/
實作
一種可以在 Minecraft 中使用 SpongeAPI 的模組或程式。
IRC
互联网中继聊天（Internet Relay Chat，缩写 IRC），可当作普通聊天平台使用。Sponge 有基于 IRC 的聊天服务器，和 Sponge 的 Discord 服务器保持同步。
問題（Issue）
在 GitHub 和其他地方標記問題、錯誤、改進和待辦事項的方法。Sponge 專案使用 GitHub 作為問題追蹤器（Issue Tracker）。
Java
Minecraft 和 Sponge 使用的程式語言，而且必需用它來執行。https://java.com/zh_TW/
MCP
Mod Coder Pack 協助開發人員為 Minecraft 伺服器和客戶端建立模組。http://www.modcoderpack.com/website
Mixins
將 Sponge 注入 Minecraft 的特製程式組件。https://github.com/SpongePowered/Mixin/wiki
Mod
一種 Minecraft 的修改，以某種方式改變遊戲玩法。使用 Forge API 編寫的模組需要 Forge 來執行，但是有一些模組可以單獨安裝。
NPC
非玩家角色。任何不受玩家控制的角色，例如村民。
Ore
Sponge 官方的插件托管网站。https://ore.spongepowered.org/
Patreon
可用於贊助專案與組織的服務。 如果你想支持 Sponge，這裡就是你的目的地！ Sponge Patreon Donation Page
插件
一個透過 SpongeAPI 來擴展和改變 Minecraft 的 Minecraft 模組，只能用於 Sponge 伺服器。
專案負責人
整個 Sponge 專案的負責人，負責協調各個小組和子專案的活動。
合併請求（Pull Request，PR）
向開放的開發專案提交貢獻的方法，例如 GitHub 上的 Sponge 儲存庫。
Sponge
整個專案的名稱。
SpongeAPI
由 Sponge 專案開發的 Minecraft 插件 API。
SpongeAuth
對於所有 Sponge 網站的認證門戶與 SSO 解決方案。 https://auth.spongepowered.org/
SpongeCommon
一個可以將程式碼提交給官方以用於建置 SpongeAPI 實作的儲存庫。SpongeVanilla 和 SpongeForge 都使用這個儲存庫。
SpongeForge
將 SpongeAPI 作為 Forge Coremod 的一項實作。
SpongeDocs
頻繁更新的專用文件，Sponge 最好的資訊來源。
Sponge 論壇
Sponge Project 的首頁，並且是 Sponge 第二好的資訊來源。定期發佈更新通知。https://forums.spongepowered.org/
Sponge 基金會
專案的財務部門，是接收、管理和分配捐贈的獨立實體。
SpongeVanilla
在原版 Minecraft（而非 Forge）上執行的 SpongeAPI 的一項實作。
Spongie
Sponge 吉祥物，一塊穿著紅披風的英勇小海綿。
Spongy
為文件提供 PR 預覽的 Sponge GitHub 機器人。
團隊領導者
領導團隊的人 (WebDev，Docs，SysOps，subreddit，IRC)。
