论坛、Discord 及 IRC 守则¶

歡迎來到 Sponge 專案！我們很高興你來到這裡，並對 Sponge 感興趣。我們有一些規則用在我們所有的各種通訊方式，包括（但不限於）：

  • 官方 IRC 頻道：#sponge，#spongedev，#spongedocs
  • Sponge 論壇
  • GitHub 與 錯誤/問題追蹤器
  • Sponge Discord
  • Sponge subreddit
  • Sponge TeamSpeak 伺服器
  • Sponge 專案的其他官方溝通方式。

Sponge 專案堅持惠頓法則（Wheaton’s Law）：「不要當個賤人。（Don’t be a dick.）」我們期望每個使用我們溝通平台的人都是講理、負責的人，並能尊重其他使用者和工作人員。

所有的使用者也應閱讀我們的 服務條款 並遵循其訂定的要求。

繼續閱讀這些更具體的指南，但請記得這些只是通用規則，而不是該做和不該做的完整清單。 這些東西應該都是一般常識。

  • 遵循支援指南（如果有）以解決您可能會有的問題。
  • 不要濫用頻道 (IRC，Forum，Discord等)。 這意味著不要發垃圾郵件，一貼多發，交叉貼或大量貼。 請記住，我們都是共享著一個相同的虛擬空間。
  • 請盡量讓用詞保持在保護級以內。我們不是你的保姆，不會為了咒罵而使你難過，但請保持理性與禮貌。
  • 禁止種族主義、性別歧視、偏見或仇恨言論。不要以身試法。嚴格禁止——不要違反這個規則。
  • 對 OP 和管理員保持友善。他們是志願者，時間往往有限。雖然他們通常很樂意與您交談，但請不要發太多私訊或是在 IRC 或 Discord 狂敲中工作人員……等。
  • 除非需要保持隱私，否則請避免私訊 ops/mods，他們不是你的個人搜索引擎。
  • 請避免討論政治、宗教和類似的爭議的話題。不僅與 Sponge 無關，往往還會引起不必要的爭執。
  • 不要抨擊其他專案、Mojang 或與之相關的人員。
  • 不要問 Sponge 什麼時候會完成。該完成的時候就會完成。完成後不要問什麼時候更新。該出來的時候就會出來。這是所有開源專案的一般規則。
  • 禁止利用 Sponge 系統進行任何形式的廣告。 這包括但不限於為你最喜歡的伺服器託管公司、你玩的伺服器、你的愛用軟體、其他遊戲以及其他事物。Off-Topic 章節中有一些例外情形，完全由工作人員決定。
  • 不要試圖在 Sponge 網站或系統上進行銷售或出售任何東西。 這包括但不限於插件（模組或任何變體），藝術資產，服務或任何提供的工作。 我們特別 允許 在 Service Exchange 區域的 原始帖子 中提供賞金，開發者可以在 Devs Available 類別中尋求委託工作。 禁止在這些（或任何其他）討論串中進行付款討論。
    • 使用者可以在「Service Exchange」類別中發佈招聘廣告（有或無報酬），但是必須遵循 Template and Rules 。
    • 使用者可以在「Devs Available」類別內發佈工作請求（有或無報酬），但是必須遵循 Template and Rules 。
    • Service Exchange 中也不允許付費插件
  • 如果我們發現由 Sponge 託管的 「lite」 版插件被用來推銷 「premium」 版，我們保留刪除插件列表的權利。」lite」 或 「premium」 將由 Sponge 人員自行判斷。
  • 你的插件名稱中請不要帶有 Sponge。我們為 SpongePowered 官方作品保留使用 Sponge 的權利。但有兩個例外：如果你的插件有其他 API 的版本，可以在標題中加上 「for Sponge」，或是主要針對 Minecraft 「海綿」 方塊的插件也可以在標題中使用 Sponge。
  • 我們的目標是完全遵守 Mojang EULA，為此，任何被懷疑違反 EULA 的插件、服務、帖子和/或連結都可以在 Sponge 工作人員的酌處下或應 Mojang AB 的請求刪除。
  • 不要散佈關於 Sponge 專案、工作人員、附屬機構或 Sponge 合作夥伴的謠言或虛假言論，或任何其他可能損害專案的誹謗言論。
  • 如果你被封鎖或被停權，一定有其原因。請不要試圖繞開封鎖。
  • Sponge 專案所使用的各種溝通管道的板主、系統管理員或管理人員在任何爭議中具有最終的決定權。

備註

以上規定中的任何一條都可以隨時更改，恕不另行通告。任何發現違反這些規則的內容，即使內容是在規則制定之前創建的，都可以由版主批准刪除。這些規則通常適用於所有 Sponge 的溝通方法，但是每個子方法（例如：subreddit 或論壇分類）都可能有其自己的一套附加規則和指導方針來添加在這些通用規則之上。如果情況正是如此，則這些規則都必須被遵循。規則和準則隨著 Sponge 社區的發展而會更改。必須按照指示使用；過度使用 FLARD 可能會「使你的襪子腐爛」。
