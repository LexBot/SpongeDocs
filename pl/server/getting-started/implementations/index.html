Wybór implementacji¶

Coś co potrafi uruchamiać pluginy Sponge jest nazywane implementacją. Dopóki plugin jest poprawnie napisany wykorzystując Sponge API, dopóki powinien poprawnie działać na dowolnej w dostatecznie kompletnej implementacji.

Minecraft domyślnie nie jest w stanie uruchamiać pluginów Sponge, ale da się go zmodyfikować tak, aby mógł.

SpongeAPI itself is an open standard.

Dostępne implementacje¶

Aktualnie istnieją dwie implementacje:

Nazwa W oparciu o
SpongeForge Mojang’s „vanilla” Minecraft: Java Edition and Minecraft Forge
SpongeVanilla Mojang’s „vanilla” Minecraft: Java Edition

Którą implementację wybrać?¶

Jeśli chcesz grać z modami Forge albo chcesz korzystać ze Sponge w singleplayer, wybierz SpongeForge.

If you only want to run a Minecraft server with plugins on it (but no mods), then you can choose SpongeForge or SpongeVanilla. SpongeForge supports vanilla clients, as long as you don’t install Forge mods which require client-side mods. If you prefer to run a server without Forge, then SpongeVanilla is your preferred option.

SpongeVanilla i SpongeForge (bez modów) działają tak samo, więc wybór między nimi jest tylko kwestią preferencji.
