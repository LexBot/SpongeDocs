Problemen oplosssen¶

U bent waarschijnlijk hier omdat er iets mis is gegaan met uw Sponge server. Laten we eens kijken of we kunnen achterhalen wat het zou kunnen zijn, en wat we er aan moeten doen.

Potential Sources of Trouble

  • Java is niet geïnstalleerd op je computer
  • Netwerk connectie fout (of DDOS aanval)
  • Niet genoeg vrij geheugen
  • Malformed Config File (e.g. Bad Editing)
  • Een plugin (of mod) werkt niet naar behoren
  • Operating System Unstable (e.g. Virus Infection)
  • Corrupte data
  • Een menselijk probleem
  • Een fout in Sponge code
  • Er is iets mis met het universum..

Java is niet geïnstalleerd op je computer¶

Solution: Get Java. Visit the Java installeren for more information.

Netwerk connectie fout (of DDOS aanval)¶

Symptomen: Netwerkverbinding is erg traag, valt constant uit of is totaal verdwenen.

Oplossingen*: Check uw internetverbinding naar de router of modem. Kijk of uw browser dezelfde problemen ondervindt. U kunt een gratis dienst zoals speedtest.net gebruiken om uw verbindingssnelheid te testen. Andere programma’s op uw computer of lokale netwerk kunnen de problemen veroorzaken. Zorg ervoor dat u Port Forwarding op uw router heeft ingeschakeld. Een DDoS-aanval, hoewel zeer onwaarschijnlijk, kan uw hele internetverbinding platleggen. U zult uw ISP moeten contacteren voor extra hulp.

Niet genoeg vrij geheugen¶

Symptomen: Server crasht, vaak met berichten over ‘Out of Memory’

Solutions: Expand the maximum Perm memory size with the startup argument -XX:MaxPermSize=128. Expand your server heap memory (if possible) with startup arguments e.g. -Xms1024M (1GB starting memory) and -Xmx2048M (2GB maximum). Monitor your free memory on the computer and see if there is some locked up in other processes. You may need to kill frozen java processes, or restart your machine. Memory leaks sometimes occur with bugs in plugins, which can take time to isolate.

Nog steeds een probleem?: Als je nog steeds problemen ondervindt, hoewel je het bovenstaande hebt geprobeerd en niet de ‘heap size’ kunt verhogen, kijk dan in de ‘Task Manager’ om te zien of je alle werkgeheugen van je computer gebruikt. Als dit het geval is, is de enige optie om meer RAM toe te voegen aan je computer. Als er nog voldoende werkgeheugen over is, betekent dit dat je de 32-bits versie van Java draait. We raden je aan om over te stappen naar de 64-bits versie, als je besturingssysteem ook 64-bit ondersteunt.

Malformed Config File (e.g. Bad Editing)¶

Symptomen: Één (of meerdere) plugins willen niet laden, of gedragen zich niet zoals ze zich zouden moeten gedragen. De server log bestanden zullen informatie bevatten over onleesbare bestanden bij het opstarten. De server kan hierdoor crashen en data kan corrupt raken.

Oplossing: Stop de server en bekijk uw bewerkte bestanden. Als er beschadigde data is, vervang deze dan met een eerder gemaakte back-up. Het is mogelijk dat u een volledig configuratie bestand moet verwijderen, zodat deze opnieuw gegenereerd kan worden bij het opstarten van de server.

Een plugin (of mod) werkt niet naar behoren¶

Symptomen: Dit kan bijna alles zijn - plugins maken soms fouten, dus kijk of dit een heel specifiek probleem is met een of meerdere plugins. Meestal gaan dit soort fouten gepaard met foutmeldingen in de server logs.

Solution: Stop the server, and check to see nothing has been corrupted. Be sure to check that it isn’t from an incorrectly edited config file (above). Remove suspect plugins and add them again one by one, restarting the server each time. The problem may originate from one plugin that is out of date - check for updates. Plugin conflict may also be the cause, having two incompatible plugins. You can also check our Mod Compatibility page to see if the problem is already known, and if there’s a way to fix it.

Operating System Unstable (e.g. Virus Infection)¶

Symptomen: De server blijft herhaaldelijk crashen en andere delen van je besturingssysteem werken niet zoals ze houden moeten.

Solution: Stop everything. Thoroughly check your system and storage devices for malware and viruses. Good tools for this include AdwCleaner, Junkware Removal Tool, MalwareBytes, and most antivirus ware. Check your server files for corruption after a clean restart of your system. Examine the hardware for damage too if the problems persist - e.g. a faulty power supply.

Corrupte data¶

Symptomen: Bestanden waar de wereld bewaard wordt, is corrupt. Soms crasht de server of spelers als ze in een bepaald deel van de wereld komen. Kan onder andere te maken hebben met een corrupte database.

Oplossing: Laad de back-up bestanden voor de corrupte data. Er zijn programma’s die beschadigde werelden fixen en missende regio’s kunnen opnieuw gegenereerd worden. Ga opzoek naar de oorzaak van de corruptie, misschien was het wel een kapotte plugin, database driver, te weinig power of iets anders? Zorg ervoor dat je hoe dan ook altijd back-ups maakt van belangrijke data op een goed beveiligde plek.

Een menselijk probleem¶

Symptomen: Gister werkte alles nog prima, maar nu, nadat ik iets aangepast heb, werkt het niet meer..

Oplossing: SpongeDocs is niet groot genoeg om te omvatten wat mensen kunnen doen waardoor software onverwacht kapot gaat. Het werkt meestal het best om te bedenken welke recente aanpassingen je hebt gedaan, die je server heeft kunnen laten crashen of stroef loopt. Zit er bijvoorbeeld een geheugen kaart los na het afstoffen van je computer…

Een fout in Sponge code¶

Symptomen: Geen enkele van de bovengenoemde symptomen en het werkt nog steeds niet zoals het zou moeten.

Oplossing: Tijd om het grof geschut erbij te halen. Dien een melding in op de SpongeForge of SpongeVanilla probleem tracker, vergeet niet om details over de versies van Forge en Sponge toe te voegen. Ook een link naar het desbetreffende logboekbestand van uw server is handig.

Er is iets mis met het universum..¶

Sorry man, we kunnen je echt niet helpen. Je zult het zelf moeten oplossen. Succes!
