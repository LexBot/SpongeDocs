Het kiezen van een implementatie¶

Iets waarop Sponge plugins worden uitgevoerd heet een implementatie. Zolang een een plugin correct is gemaakt met behulp van de SpongeAPI, moet deze werken op elke voldoende complete implementatie.

Minecraft kan Sponge plugins niet uitvoeren, maar u kunt haar zo veranderen dat zij het wel kan.

SpongeAPI itself is an open standard.

Beschikbare Implementaties¶

Er zijn momenteel twee implementaties:

Naam Gebaseerd op
SpongeForge Mojang’s “vanilla” Minecraft: Java Edition en Minecraft Forge
SpongeVanilla Mojang’s “vanilla” Minecraft: Java Edition

Welke moet ik kiezen?¶

Als u MinecraftForge mods of Sponge in singeplayer wilt gebruiken , kies dan voor SpongeForge.

If you only want to run a Minecraft server with plugins on it (but no mods), then you can choose SpongeForge or SpongeVanilla. SpongeForge supports vanilla clients, as long as you don’t install Forge mods which require client-side mods. If you prefer to run a server without Forge, then SpongeVanilla is your preferred option.

SpongeVanilla en SpongeForge (zonder mods) gedragen zich hetzelfde, dus de beslissing tussen de twee is een kwestie van voorkeur, niet een keuze van functionaliteit of features.
