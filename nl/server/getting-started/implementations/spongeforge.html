SpongeForge installeren¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it’s more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Gebruikers die geen Minecraft Forge willen gebruiken kunnen overwegen om SpongeVanilla te gebruiken.

Downloaden¶

Review our Versioning and grab your copy of SpongeForge here.

SpongeForge installeren¶

Notitie

Als u een game server host gebruikt (of van plan bent te gaan gebruiken), hebben ze wellicht een configuratiescherm dat Sponge voor u kan installeren.

Notitie

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_74. Note that Sponge requires at least 1.8.0_20 or above to run properly but it is recommended to use the latest Java 8 version.

Een speler / In-game LAN servers¶

  1. Download het installatieprogramma van Minecraft Forge via de Minecraft Forge website. Zorg ervoor dat u exact hetzelfde build-nummer gebruikt als hierboven is weergegeven.
  2. Voer het installatieprogramma van Forge uit. Een nieuw Forge profiel zal aangemaakt worden in de Minecraft launcher.
  3. Open de Minecraft launcher en selecteer het nieuwe Forge profiel.
  4. Klik op “Options” en klik dan op “Open Game Dir”.
  5. Download SpongeForge van de Sponge website en sleep in het mods mapje. Maak deze map aan mocht deze nog niet bestaan.
  6. Sponge moet zowel in single player als in een naar LAN geopende wereld werken.

Leer vervolgens hoe u Sponge configureert en hoe u uw instantie van Sponge beheert (inclusief het installeren van plugins).

Dedicated servers¶

Notitie

Als u al een Forge server heeft kunt u de Sponge mod direct in uw mods mapje plaatsen. Vergeet niet om Forge te updaten zodat de versie overeenkomt met de versie die SpongeForge nodig heeft. Neem een kijken bovenaan deze pagina als u niet zeker weet welke versie u nodig heeft.

Forge via de opdrachtregel installeren¶

  1. Bezoek de Minecraft Forge website en klik op “Show all downloads” om een volledige lijst met beschikbare versies te bekijken. Zoek de versie die overeenkomt met de versie in de bestandsnaam van de SpongeForge download, en beweeg over de (i) naast “Installer” om een directe download link te verkrijgen.
  2. Gebruik uw favoriete download-methode om het jar bestand naar de juiste locatie te downloaden. Bijvoorbeeld: wget https://url.to/forge-version-installer.jar
  3. Vanuit de map waarin u Forge wilt installeren moet u het jar bestand uitvoeren met de optie --installServer. Bijvoorbeeld: java -jar forge-version-installer.jar --installServer
  4. Ga verder naar ‘SpongeForge toevoegen aan Forge’ hieronder.

Forge installeren via GUI¶

  1. Download het Minecraft Forge installatieprogramma, waarvan de versie overeenkomt met de versie die in de bestandsnaam van SpongeForge staat, via de Minecraft Forge website. Zie hierboven voor het schema voor de naamgeving van SpongeForge en Forge.
  2. Voer het Forge installatiebestand uit, klik op ‘Install Server’, kies een lege map om de bestanden van de server te plaatsen en klik vervolgens op ‘OK’.
  3. Ga verder naar ‘SpongeForge toevoegen aan Forge’ hieronder.

SpongeForge toevoegen aan Forge¶

  1. Download SpongeForge van de Sponge website en stop deze in het mods mapje van uw server. Maak deze map aan mocht deze nog niet bestaan.
  2. U kunt nu de server opstarten met behulp van het commando of opstartscript java -jar forge-version-XYZ.jar.
  3. Als u vanuit thuis werkt vergeet dan om Port forwarding in te stellen, zodat anderen kunnen verbinden met uw server.

Leer vervolgens hoe u een opstartscript maakt en gebruikt, Sponge configureert en uw server beheert (inclusief het installeren van plugins).
