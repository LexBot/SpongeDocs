Een opstart script maken¶

Notitie

Deze instructies gelden alleen voor u als u uw Minecraft server wilt uitvoeren op uw eigen machine. De meeste Minecraft hosts maken zelf een lancering script voor u aan.

Het schrijven van een opstart script¶

Open eerst een teksteditor zoals Atom, Sublime Text of Kladblok. Schrijf (of plak) een opstartscript voor uw server. Voorbeelden van eenvoudige opstartscripten voor Windows, macOS en Linux zijn hieronder verstrekt. Houd de RAM beperkingen van uw machine in gedachten.

Notitie

The following examples are generic. For a Forge server using Sponge (coremod), change forge-1.12.2-XYZ-universal.jar to whatever your Forge version in the server directory is named. To launch a SpongeVanilla server, change forge-1.12.2-XYZ-universal.jar to the name of the SpongeVanilla.jar file.

Windows¶

java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar
pause

Sla uw Windows opstart script op als launch.bat.

Waarschuwing

Launching your server in a Command Prompt or PowerShell environment can lead to the server hanging due to the Quick Edit Mode of these shells. This mode freezes the process when you highlight something or click inside the console window. During this time, messages captured in the log will state that the server skipped many seconds or minutes worth of ticks. You can prevent this from occurring by not highlighting any text on the screen and not clicking inside the window, or by disabling the Quick Edit Mode in the Properties dialogue.

macOS¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Sla uw Mac opstart script op als launch.command.

Linux¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Sla uw Linux opstart script op als launch.sh.

Het uitvoeren van een opstart script¶

Zorg ervoor dat u uw opstart script uitvoert buiten de map speciaal voor de server.

U kunt uw opstart script uitvoeren door hier dubbel op te klikken. Als u van een console of terminal gebruikt maakt, navigeer naar de map van het script en voer het uit. Onthoud dat u nog akkoord moet gaan met de Mojang EULA voordat u de server kunt uitvoeren.

Notitie

De standaard Minecraft server GUI-console is uitgeschakeld door Sponge, omdat deze erg processor intensief is.

Waarschuwing

Als u een permissie fout krijgt terwijl u uw server probeert te starten op een Mac, probeer dit:

  • Open de Terminal.
  • Type chmod a+x, met een spatie aan het einde.
  • Sleep uw opstart script naar de Terminal.
  • Druk op enter.
