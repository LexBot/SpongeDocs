Een server aanmaken¶

The articles in this section explain how to create and run a server powered by an implementation of SpongeAPI.

Als u een gebruiker bent die een nieuwe server wilt maken met behulp van Sponge of een bestaande server wilt migreren naar een implementatie van Sponge, dan is dit voor u. Plugin ontwikkelaars zijn wellicht ook geïnteresseerd in de artikelen binnen deze sectie - test servers kunnen namelijk nuttig zijn voor de ontwikkeling van een plugin.

Notitie

We attempt to be implementation-agnostic when possible, meaning that instructions generally aren’t specific to one implementation of SpongeAPI. Nonetheless, mistakes can happen.
