Sponge woordenlijst¶

Dit is een gids voor de meest voorkomende gereserveerde woorden, zinnen en afkortingen die gebruikt worden in het Sponge Project. Snelkoppelingen zijn in sommige gevallen toegevoegd om meer informatie te verstrekken. Het is niet bedoeld als een gids voor SpongeAPI of Java.

Notitie

Termen zijn gedefinieerd onder verwijzing naar Minecraft en/of het Sponge project zoals vereist is.

Definities¶

API
Application Programming Interface - een manier om modding gemakkelijker te maken.
Bijdrager
Iemand die code bijdraagt aan de SpongePowered projecten.
Coremod
Een Coremod is een Forge mod die wordt geladen voordat de klassen van Mojang worden geladen, zodat de code van Minecraft kan worden veranderd via Access transformatoren, ASM klasse aanpassingen of andere Java bytecode manipulaties.
Crowdin
Een web-gebaseerd vertaal systeem voor de documentatie. SpongeDocs gebruikt het om vertaalde documenten beschikbaar te stellen in vele talen. https://crowdin.com/project/sponge-docs
Ontwikkelaar
Iemand die programmeert (Sponge, Plugins, Mods, …).
Discord
Discord is a free voice, video and text chat app, and is used as a chat platform. Like IRC, Provides chat rooms for all things Sponge. Sponge’s Discord server is synced with Sponge’s IRC server.
FLARD
Het is groen, eetbaar en explosief. Laat het alleen en het zal u alleen laten.
Forge
Een al lang bestaande Minecraft modding API, met een eigen installatieprogramma. Met Forge kunt u mods in single-player en op LAN & dedicated servers gebruiken. https://mcforge.readthedocs.io/en/latest/
GitHub
Een gratis openbare web-gebaseerd systeem voor het delen van code en het creëren van projecten. https://github.com/
Implementatie
Een mod of programma dat het gebruik van SpongeAPI in Minecraft mogelijk maakt.
IRC
Internet Relay Chat, used as a chat platform. Provides chat rooms for all things Sponge. Sponge’s IRC server is synced with Sponge’s Discord server.
Probleem
Een manier om problemen, bugs, verbeteringen en dingen om te doen op GitHub en elders te markeren.
Java
De programmeertaal waar Minecraft en Sponge in zijn geschreven en nodig hebben om te kunnen werken. https://java.com/en/
MCP
De Mod Coder Pack helpt ontwikkelaars met het maken van mods voor zowel de Minecraft client als de server. http://www.modcoderpack.com/website
Mixins
Gespecialiseerde programmaonderdelen die Sponge in Minecraft injecteren. https://github.com/SpongePowered/Mixin/wiki
Mod
Een Minecraft modificatie dat op een een of andere manier iets verandert aan de gameplay. Mods geschreven met behulp van de Forge API hebben Forge nodig om te kunnen werken, maar sommige mods kunnen ook zonder Forge geïnstalleerd worden.
NPC
Non-Player Character. Elk karakter dat niet wordt beheerst door de speler, bijv. een dorpeling.
Ore
De officiële Sponge plugin hosting website. https://ore.spongepowered.org/
Patreon
Een dienst voor donaties aan projecten en organisaties. Als u Sponge wilt ondersteunen is dit de plek om te kijken! Sponge Patreon Donatie Pagina
Plugin
Een Minecraft mod die Minecraft uitbreidt en verandert via SpongeAPI, alleen te gebruiken op Sponge servers.
Projectleiders
De mensen die de leiding hebben over het Sponge project, verantwoordelijk voor de coördinatie van de activiteiten van de diverse teams en sub-projecten.
Pull Request (PR)
Een methode voor het indienen van bijdragen aan een open ontwikkelingsproject, zoals de Sponge opslagplaatsen op GitHub.
Sponge
De naam van het gehele project.
SpongeAPI
De Minecraft Plugin API ontwikkeld door het Sponge project.
SpongeAuth
Het authenticatiepaneel en de SSO-oplossing voor alle websites van Sponge. https://auth.spongepowered.org/
SpongeCommon
Een opslagplaats voor code die de officiële SpongeAPI-implementaties gemeen hebben, gebruikt om deze implementaties te bouwen. SpongeVanilla en SpongeForge gebruiken deze opslagplaats.
SpongeForge
De implementatie van de SpongeAPI als een Forge Coremod.
SpongeDocs
Toegewijde en regelmatig bijgewerkt documentatie, de beste bron van informatie over Sponge.
Sponge Forums
De thuisbasis van het Sponge Project en de op een na beste bron van informatie over Sponge. Update-aankondigingen worden regelmatig gepubliceerd. https://forums.spongepowered.org/
Sponge Foundation
De financiële kant van het project, een aparte entiteit die donaties accepteert, beheert en distribueert.
SpongeVanilla
Een implementatie van SpongeAPI die draait op Vanilla Minecraft (en niet op Forge).
Spongie
De mascotte van Sponge, een kleine moedige spons met een rode cape.
Spongy
De Sponge GitHub Bot die PR previews maakt voor de documentatie.
Teamleiders
De mensen die de teams leiden (WebDev, Docs, SysOps, subreddit, IRC).
