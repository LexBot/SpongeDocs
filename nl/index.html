Sponge Documentatie¶

Inleiding¶

Welkom bij SpongeDocs, de officiële documentatie voor het open-source Sponge project.

SpongeAPI is your all new plugin API for Minecraft: Java Edition. You can run it on a plain vanilla server (as SpongeVanilla) or on your fully modded MinecraftForge server (as SpongeForge). Check out our Versiebeleid to understand selecting a file and grab your copy from our downloads page.

Niet zeker van wat u zoekt? Probeer onze Over het Sponge project sectie, waarin een kleine Inleiding, onze Veelgestelde vragen en een artikel over De structuur van het Sponge Project staan.

Server eigenaren zouden onze tips en instructies moeten lezen in Een server aanmaken over hoe je een Sponge gebaseerde server installeert, uitvoert en onderhoudt.

Het Sponge project heeft verschillende hoofd onderdelen. Leer meer over de structuur van het project hier of bezoek hun repositories op GitHub:

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore | SpongeAuth

Notitie

Sponge development is ongoing. Stable and Experimental builds of both SpongeForge and SpongeVanilla are currently available for download.
