Създаване на сървър¶

The articles in this section explain how to create and run a server powered by an implementation of SpongeAPI.

Ако сте потребител, който желае да създаде нов сървър с помощта на реализация или да мигрира съществуващ сървър към реализация, това е за вас. Разработчиците на приставки може също да се заинтересуват от статии в раздела - пробните сървъри могат да са полезни за разработката.

Note

We attempt to be implementation-agnostic when possible, meaning that instructions generally aren’t specific to one implementation of SpongeAPI. Nonetheless, mistakes can happen.
