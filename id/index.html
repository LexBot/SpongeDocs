Sponge Documentation¶

Pengantar¶

Selamat datang di SpongeDocss, dokumentasi resmi untuk open-source Sponge project.

SpongeAPI is your all new plugin API for Minecraft: Java Edition. You can run it on a plain vanilla server (as SpongeVanilla) or on your fully modded MinecraftForge server (as SpongeForge). Check out our Versioning Policy to understand selecting a file and grab your copy from our downloads page.

Tidak yakin dengan apa yang sedang anda cari? Cobalah bagian Tentang Sponge Project, yang berisi Pengantar secara singkat, Pertanyaan Yang Sering Diajukan milik kami dan artikel tentang Struktur dari proyek Sponge.

Pemilik Server harus membaca tips dan petunjuk dalam :doc:`server/index`bagaimana untuk menginstal, menjalankan dan mempertahankan Sponge berbasis server.

Sponge project memiliki beberapa komponen utama. Pelajari lebih lanjut mengenai struktur proyek here atau kunjungi repositori mereka di GitHub:

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore | SpongeAuth

Catatan

Sponge development is ongoing. Stable and Experimental builds of both SpongeForge and SpongeVanilla are currently available for download.
