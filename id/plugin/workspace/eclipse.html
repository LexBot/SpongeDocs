Menyediakan eclipse¶

Artikel ini menggambarkan bagaimana caranya anda untuk mengkonfigurasikan Eclipse ruang pekerjaan bagi perkembangan plugin dengan SpongeAPI dan membangun sebuah sistem yaitu seperti Maven atau Gradle.

Catatan

Sebuah tutorial oleh Mumfrey memperlihatkan pengaturan dari sebuah ruang pekerjaan Sponge pada Eclipse, menggunakan fitur terbaru pada Pembangunankapal dan MenempaGradle, bisa dilihat disini.

Gradle¶

Mempergunakan gradle¶

Catatan

Gradle ini secara otomatis termasuk di dalam Eclipse memulai dari pelepasan Neon (22 Juni 2016) dan nanti. Langkah ini hanya dibutuhkan untuk versi tedahulu.

Anda harus memasangkan terlebih dahulu Gradle Integrasi Plugin sebelum mempergunakan Gradle pada Eclipse. Ini hanya kebutuhan yang harus dilakukan berdasarkan penciptaan proyek pertama anda.

Tip

Biasanya, anda tidak membutuhkan Melompati modul opsional yang dibagikan dengan plugin ini, jadi anda tidak bisa bisa menghapus centang selama instalasi.

  • Buka ** Eclipse **.
  • Klik `` Help> Eclipse Marketplace``.
  • Cari `` Gradle Integration Plugin ``.
  • Instal Plugin Plugin ** Gradle **.

Anda membuat proyek¶

  • Buka ** Eclipse **.
  • Klik `` File> Baru> Lainnya``.
  • Pilih `` Gradle> Gradle Project``.
  • Klik ' ' selanjutnya ' '.
  • Masukkan nama proyek, kemudian klik Finish.

Mengedit script build¶

  • Buka membangun.gradle di navigator.
  • Edit skrip pembangunan sesuai dengan intruksi pada Menyiapkan Gradle.
  • Klik-kanan pada proyek anda, lalu pilih Gradle > Segarkan Proyek Gradle.

Anda mengimpor proyek¶

Jika anda sudah mulai dengan proyek anda dan ingin mengimpor lagi pada suatu titik kemudian anda perlu impor itu bukan menciptakan kembali itu dalam IDE anda:

  • Klik `` File> Impor``
  • Pilih `` Gradle> Gradle Project``
  • Arahkan ke folder root dari proyek
  • Klik selesai

Maven¶

Anda membuat proyek¶

  • Buka ** Eclipse **.
  • Klik `` File> Baru> Lainnya``.
  • Pilih Maven pada sisi kiri popup, dan pilih Maven Project di sisi kanan.
  • Klik ' ' selanjutnya ' '.
  • Pilih membuat sebuah proyek sederhana, kecuali anda memerlukan lebih advanced setup.
  • Masukkan ID grup anda ID Grup, Artefak ID, nama proyek, dan diskripsi proyek.
    • ** ID Grup Anda ** biasanya harus sesuai dengan nama paket Java Anda. Lihat : doc: ../ plugin-class untuk detailnya.
    • ID Artifact ** ** Anda biasanya harus sesuai dengan ** plugin ID ** yang Anda pilih sebelumnya, misalnya `` myplugin``.
    • Your Version is up to you. See Version Numbers for details.
  • Klik Selesai.

Tip

Gerhana sering tidak membuka proyek-proyek baru setelah menciptakan mereka. Jika ini terjadi, cobalah menutup Gerhana layar selamat datang; proyek anda harus terbuka di setelah itu.

Sedang mengedit konfigurasi proyek¶

  • Buka pom.xml pada navigator.
  • Edit konfigurasi pembangunan sesuai dengan intruksi pada Mempersiapkan Maven.
  • Segarkan maven project.

Anda mengimpor proyek¶

Jika anda sudah mulai dengan proyek anda dan ingin mengimpor lagi pada suatu titik kemudian anda perlu impor itu bukan menciptakan kembali itu dalam IDE anda:

  • Klik `` File> Impor``
  • Pilih Maven > Ada Proyek Maven
  • Arahkan ke folder root dari proyek
  • Klik selesai
