Membuat sebuah server¶

The articles in this section explain how to create and run a server powered by an implementation of SpongeAPI.

Jika anda adalah seorang pengguna yang ingin membuat sever baru menggunakan implementasi dari Sponge atau perpindahan server yang tersedia kepada implementasi dari Sponge, inilah untuk anda. Pengembang plugin bisa saja tertarik pada artikel yang tersedia dalam bagian ini - tes server bisa saja beguna bagi pengembang plugin.

Catatan

We attempt to be implementation-agnostic when possible, meaning that instructions generally aren't specific to one implementation of SpongeAPI. Nonetheless, mistakes can happen.
