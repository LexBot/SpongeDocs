Pemecahan masalah¶

Anda mungkin di sini karena ada sesuatu yang salah dengan anda Sponge server. Mari kita lihat jika kita dapat mencari tahu apa itu, dan apa yang harus dilakukan tentang hal itu.

Potential Sources of Trouble

  • Jawa Tidak Diinstal Pada Komputer Anda
  • Koneksi jaringan Gagal (or DDoS Attack)
  • Tidak Cukup Memori Bebas
  • Malformed Config File (e.g. Bad Editing)
  • Sebuah Aplikasi tambahan (atau Mod) mengalami kegagalan fungsi
  • Operating System Unstable (e.g. Virus Infection)
  • Data Yang Rusak
  • Masalah Antara Keyboard dan Kursi
  • Ada Bug di Spons
  • Ada Sesuatu yang Salah Dengan alam Semesta

Jawa Tidak Diinstal Pada Komputer Anda¶

Solution: Get Java. Visit the Menginstalasi Java for more information.

Koneksi jaringan Gagal (or DDoS Attack)¶

Gejala: koneksi jaringan sangat lamban, timbul tenggelam, atau tidak ada sama sekali.

Solusi: Periksa koneksi anda ke modem atau router. Lihat jika browser anda memiliki masalah serupa. Anda dapat menggunakan layanan gratis seperti speedtest.net untuk memeriksa kecepatan koneksi anda. Layanan lain yang berjalan pada komputer anda atau jaringan lokal dapat menyebabkan. Pastikan bahwa anda telah mengaktifkan Port Forwarding di router anda. Serangan DDoS, sementara tidak mungkin, mungkin akan benar-benar membunuh konektivitas anda, dan anda harus menghubungi ISP anda jika anda percaya hal ini akan terjadi.

Tidak Cukup Memori Bebas¶

Gejala: Peladen rusak, biasanya juga disertai dengan pesan ''Memory tidak Cukup''.

Solutions: Expand the maximum Perm memory size with the startup argument -XX:MaxPermSize=128. Expand your server heap memory (if possible) with startup arguments e.g. -Xms1024M (1GB starting memory) and -Xmx2048M (2GB maximum). Monitor your free memory on the computer and see if there is some locked up in other processes. You may need to kill frozen java processes, or restart your machine. Memory leaks sometimes occur with bugs in plugins, which can take time to isolate.

Masih masalah?: Jika anda masih mengalami masalah meskipun di atas dan tidak bisa meningkatkan Ukuran Tumpukan, cek di Task Manager anda untuk melihat jika anda menggunakan semua Memori yang tersedia. Jika anda, satu-satunya solusi adalah untuk menambahkan lebih banyak RAM untuk sistem anda. Jika masih ada banyak memori yang tersedia, anda menjalankan 32-bit Jawa. Jika anda menggunakan 32-bit Jawa, kami sarankan upgrade ke 64-bit Jawa, disediakan bahwa Sistem Operasi anda juga 64-bit.

Malformed Config File (e.g. Bad Editing)¶

Gejala: Satu aplikasi tambahan (atau lebih) menolak untuk dimuat, atau berperilaku tidak biasanya. Berkas kunci peladen akan berisi pesan-pesan tentang berkas-berkas yang tidak bisa dibaca ketika memulai. Peladen mungkin macet, dan data mungkin rusak.

**Solusi*: Matikan peladen, dan periksa berkas anda yang telah tersunting. Jalankan berkas cadangan dari data yang rusak. Anda mungkin perlu menghapus semua konfigurasi berkas dan izinkan berkas tersebut diperbaiki saat peladen dijalankan.

Sebuah Aplikasi tambahan (atau Mod) mengalami kegagalan fungsi¶

Gejala: Ini bisa jadi apa saja - apapun yang dilakukan aplikasi tambahan anda, ditambah faktor X. Biasanya peladen jadi macet dengan pesan-pesan kesalahan yang banyak didalam berkas kunci peladen.

Solution: Stop the server, and check to see nothing has been corrupted. Be sure to check that it isn't from an incorrectly edited config file (above). Remove suspect plugins and add them again one by one, restarting the server each time. The problem may originate from one plugin that is out of date - check for updates. Plugin conflict may also be the cause, having two incompatible plugins. You can also check our Mod Compatibility page to see if the problem is already known, and if there's a way to fix it.

Operating System Unstable (e.g. Virus Infection)¶

Gejala: Peladen secara terus menerus macet atau berhenti, dan bagian lain dari sistem operasi anda juga mengalami masalah.

Solution: Stop everything. Thoroughly check your system and storage devices for malware and viruses. Good tools for this include AdwCleaner, Junkware Removal Tool, MalwareBytes, and most antivirus ware. Check your server files for corruption after a clean restart of your system. Examine the hardware for damage too if the problems persist - e.g. a faulty power supply.

Data Yang Rusak¶

Gejala: Tempat berkas gagal memuat atau menyebabkan peladen macet saat pengguna memasukkan potongan tertentu. Database rusak.

Solusi: Jalankan berkas cadangan dari data yang rusak. Perangkat lunak untuk memperbaiki kerusakan dunia tersedia, dan bagian yang hilang akan di kembalikan. Cari tahu penyebab kerusakan - apakah karena colokan tidak berfungsi, kandar basis data, kegagalan daya atau sesuatu yang lain? Pastikan anda selalu membuat cadangan secara berkala data-data yang penting kedalam perangkat yang aman.

Masalah Antara Keyboard dan Kursi¶

Gejala: Semuanya lancar kemaren. Hari ini jadi aneh setelah saya melakukan XYZ...

Solusi: SpongeDocs tidak cukup besar untuk menjaring sesuatu yang mungkin dilakukan seseorang yang akan mengakibatkan kegagalan perangkat lunak yang tidak diharapkan. Selalu lebih baik berfikir panjang dan keras tentang apa yang mungkin telah anda lakukan baru-baru ini yang mungkin telah mempengaruhi kinerja peladen anda yang lancar. Sebuah kartu memori bisa kelonggaran setelah berdebu, sebuah potongan berkas bisa jadi rusak...

Ada Bug di Spons¶

Gejala: Tidak ada satu pun dari yang diatas bekerja, dan sampai sekarang masih belum bekerja sebagaimana mestinya.

Solusi: Saatnya menngeluarkan senjata yang besar. Buatkan laporan ke SpongeForge atau SpongeVanilla pelacak masalah, ingat untuk menyertakan rincian versi Forge dan Sponge yang anda gunakan, dan tautan ke berkas kunci peladen yang sesuai.

Ada Sesuatu yang Salah Dengan alam Semesta¶
