Memasang SpongeForge¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it's more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Pengguna yang tidak ingin menggunakan Minecraft Forge dapat mempertimbangkan SpongeVanilla.

Unduhan¶

Review our Versioning and grab your copy of SpongeForge here.

Memasang SpongeForge¶

Catatan

Jika anda menggunakan (atau berencana untuk menggunakan) server permainan tuan rumah, mereka mungkin memiliki panel kontrol yang dapat menginstal Spons untuk anda.

Catatan

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_74. Note that Sponge requires at least 1.8.0_20 or above to run properly but it is recommended to use the latest Java 8 version.

Pemain Solo / Server LAN dalam Permainan¶

  1. Unduh pemasangan Forge Minecraft dari website Forge Minecraft. Pastikan untuk menggunakan nomor buatan yang sama persis sesuai yang ditampilkan diatas.
  2. Menjalankan disediakan Forge installer. Baru Forge profil yang akan dibuat di Minecraft launcher.
  3. Buka Minecraft launcher, dan pilih new Forge profil.
  4. Klik "Options" dan Klik "Open Game Dir".
  5. Unduh SpongeForge dari website Sponge dan letakkan kedalam berkas mods. Buatlah berkas baru jika belum ada.
  6. Spons harus bekerja baik dalam single player dan jika anda membuka dunia untuk LAN.

Selanjutnya, pelajari bagaimana anda dapat mengatur Sponge dan bagaimana untuk :doc:` mengatur contoh Sponge anda </server/management/index>` (termasuk pemasangan plugin).

Dedicated Servers¶

Catatan

Jika anda sudah memiliki Menempa sebuah server, hanya menempatkan Spons mod ke mods folder. Ingat untuk memperbarui anda Forge versi untuk mencocokkan satu yang SpongeForge membutuhkan. Lihat di bagian atas halaman ini jika anda tidak yakin versi mana yang anda butuhkan.

Menginstal Forge melalui Commandline¶

  1. Kunjungi website Forge Minecraft dan klik "Tampilkan semua unduhan" untuk melihat seluruh opsi yang tersedia. Cari tahu versi yang sesuai dengan yang didaftarkan :doc:` didalam nama file dari unduhan SpongeForge <spongeforge>, dan arahkan kursor ke (i) didekat "Installer" untuk mendaptkan tautan unduhan secara langsung.
  2. Gunakan metode unduhan kesukaan anda untuk mengunduh jar kedalam tempat tujuannya. Contohnya: wget http://url.to/forge-version-installer.jar
  3. Dari berkas dimana anda ingin memasang Forge, jalankan jar dengan pilihan --installServer. Contohnya: java -jar forge-version-installer.jar --installServer
  4. Lanjutkan untuk Menambahkan SpongeForge untuk Menempa di bawah ini.

Menginstal Forge via GUI¶

  1. Unduh pemasang Forge Minecraft dari situs Forge Minecraft untuk versi yang sesuai dengan yang terdaftar didalam nama file dari unduhan SpongeForge. Lihat keatas untuk pola penamaan dari SpongeForge dan Forge.
  2. Menjalankan disediakan Forge installer, pilih "Install Server", pilih folder kosong untuk menempatkan server's file, dan kemudian klik OK.
  3. Lanjutkan untuk Menambahkan SpongeForge untuk Menempa di bawah ini.

Menambahkan SpongeForge untuk Menempa¶

  1. Download SpongeForge dari Spons website dan memasukkannya ke dalam mods folder dalam direktori server. Membuat folder jika belum ada.
  2. Anda sekarang bisa menjalankan server melalui perintah atau script peluncuran java -jar forge-version-XYZ.jar.
  3. Jika dijalankan dari rumah, atur Meneruskan Pangkalan untuk memastikan yang lainnya bisa terhubung.

Selanjutnya, pelajari bagaimana anda dapat membuat dan menggunakan sebuah launch-script, mengatur Sponge dan atur server anda (termasuk memasang plugins).
