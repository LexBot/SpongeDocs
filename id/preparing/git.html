Memasang Git¶

Git adalah sebuah versi open-source dari pengaturan sistem, dan sangat membantu dengan pengembangan kolaborasi proyek.

The Sponge project, alongside thousands of other open-source projects, hosts its Git repositories on GitHub. Thus, Git is a crucial tool in the development of Sponge and SpongeAPI.

Git website memiliki dokumentasi yang banyak, dan laman unduhan mereka menawarkan rentang pilihan untuk klien GUI yang sesuai untuk berbagai sistem operasi.

Unduhan¶

Windows¶

GitHub Desktop adalah sebuah metode yang mudah dalam memasang Git pada Windows, karena Git termasuk dalam bagian software.

Menjalankan ulang komputer anda setelah pemasangan Git sangat dianjurkan.

Mac¶

Ada beberapa cara untuk memasang Git pada macOS.

Metode termudah adalah dengan memasang Xcode Command Line Tools.

Peringatan

Petunjuk ini tidak dapat bekerja pada Mac yang menjalankan versi lebih tua dari Mavericks. Jika anda sedang menjalankan versi dari OS X yang lebih tua dari Mavericks, lebih baik pasang klien GitHub.

  1. Luncurkan Terminal nya.
  2. Jalankan xcode-select --install.
  3. Pasangkan itu, dan ambilah kue ketika anda menunggu.
  4. Jalankan git dari Terminal.

Sebagai alternatif, anda dapat memasang Desktop GitHub. Git tersedia sebagai bagian dari pemasangan Desktop GitHub.

Menjalankan ulang komputer anda setelah pemasangan Git sangat dianjurkan.

Linux dan Unix¶

Metode termudah dalam memasang Git pada Linux adalah dengan menggunakan pengelola paket yang datang bersama penyaluran Linux anda.

Catatan

Anda mungkin harus menaruh perintah ini dengan sudo.

  1. Luncurkan Terminal nya.
  2. Run apt-get install git if you are on an Ubuntu or Debian-based distribution. Run yum install git if you are on Fedora.

Desktop GitHub saat ini sedang tidak tersedia untuk Linux, tidak seperti Windows atau Mac.

Menjalankan ulang komputer anda setelah pemasangan Git sangat dianjurkan.

Pengaturan¶

Siapakah Anda?¶

Sebelum anda memulai pekerjaan dengan git dan gudang penyimpanannya, pastikan pengaturan git anda telah mengatur identitas milik anda. Buka CLI anda dan masukkan:

git config --list

Carilah nama.pengguna dan email.pengguna. Jika mereka bukan nama pengguna dan email yang sama seperti akun GitHub anda maka ubah mereka:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Tip

Jangan memulai sumbangan apapun di bagian gudang penyimpanan Sponge manapun sampai anda membuat nama.pengguna dan email.pengguna.
