Rencana untuk masa yang akan datangÂ¶

Ada banyak hal yang ingin kami lakukan. Khususnya:

  • If Mojang releases an official modding API, we are interested in building SpongeAPI on top of Mojang's modding API so Sponge plugins will work on both.
  • Kami bisa membantu client-side Sponge mods yang dapat melakukan sebuah variasi fungsi yang berhubungan dengan client.

Ada juga beberapa hal yang kami putuskan untuk tidak dilakukan:

  • Kami tidak akan mendukung plugin Bukkit secara asli.
  • Kami tidak akan mengirimkan Forge mods dari server ke klien karena masalah keamanan.
  • Kami tidak akan mengembangkan implementasi menggunakan program bahasa yang lain, tapi kami meyakinkan orang lain untuk melakukanya. Fokus kami adalah Java.
