Panduan Forum Posting¶

Baca Sebelum Posting¶

Jika Anda bingung atau penasaran tentang Sponge, kami sangat menyarankan membaca :doc: 'faq'.

Tolong hentikan dari memposting topik yang meminta informasi dasar tentang Sponge, seperti "Apakah Sponge sebuah Forge mod", "Kapan Sponge akan dirilis", dan lain-lain. Jawaban atas pertanyaan tersebut dapat ditemukan di sini di SpongeDocs.

Sebelum Anda posting suatu topik baru di forum Sponge, pastikan tidak berlebihan dalam pertanyaan atau informasi! Menggunakan fitur pencarian (atas kanan halaman) atau periksa '' topik Anda mirip...'' kotak yang muncul di atas panel ketika membuat sebuah topik baru (ia akan mempembaharui saat Anda menambahkan sesuatu ke topik Anda). Jika ada topik yang sama, Anda mungkin harus melanjutkan diskusi Anda di sana!

Diskusi tentang API lainnya diperbolehkan selama tidak berubah menjadi buruk.

Masih perlu informasi lebih? Silahkan Lihat sumber Sponge di :doc: ' / indeks ', periksa ke 'Pengumuman Forum <https://forums.spongepowered.org/c/announcements>'__, atau untuk melacak kemajuan Sponge, 'ikuti mereka di Twitter <https://twitter.com/SpongePowered>' __ atau nonton 'Sponge GitHub <https://github.com/SpongePowered>'__. Jika Anda benar-benar tertarik untuk membantu pengembangan, lihat di :doc: '/contributing/index'.

Mengkategorikan Posting Anda¶

When creating new topics, use the drop-down menu to the right of the topic title to set its category. Try to make sure this category fits what you're posting about. If you don't know what a category implies, a description is next to each one in the drop down.

Terima kasih untuk membaca & selamat datang Spons masyarakat!
