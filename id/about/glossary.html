Daftar istilah sepon¶

This is a guide to the more common reserved words, phrases and abbreviations used in the Sponge Project. Links are provided in some cases to provide more information. It is not intended to be a guide to SpongeAPI or Java.

Catatan

Persyaratan didefinisikan dengan mengacu pada Minecraft dan/atau Proyek Sepon sesuai kebutuhan.

Definisi¶

API
Antarmuka Pemrograman Aplikasi - cara membuat modding lebih mudah.
Penyumbang
Seseorang yang menyumbang kode untuk proyek SpongePowered.
Coremod
CoreMod adalah menempa mod yang dimuat sebelum kelas Mojang dimuat, memungkinkan memodifikasi kode Minecraft asli melalui Akses transformer, modifikasi kelas ASM, atau manipulasi bytecode Java lainnya.
Crowdin
Sistem penerjemahan berbasis web untuk dokumentasi. SpongeDocs menggunakannya untuk menyediakan Dokumen yang diterjemahkan dalam banyak bahasa. https://crowdin.com/project/sponge-docs
Pengembang
Seseorang yang memprogram kode (Sepon, Plugin, Modifikasi, ...).
Discord
Discord is a free voice, video and text chat app, and is used as a chat platform. Like IRC, Provides chat rooms for all things Sponge. Sponge's Discord server is synced with Sponge's IRC server.
FLARD
Warnanya hijau, mudah dimakan, dan mudah meledak. Biarkan saja dan itu akan meninggalkan anda sendiri.
Menempa
API modding Minecraft yang telah lama berdiri, dengan memasang sendiri. Forge memungkinkan modifikasi dalam pemain tunggal, LAN, dan berdedikasi server. https://mcforge.readthedocs.io/en/latest/
GitHub
Sistem berbasis web publik gratis untuk berbagi kode dan membuat proyek. https://github.com/
Pelaksanaan
A mod or program that enables the use of SpongeAPI in Minecraft.
IRC
Internet Relay Chat, used as a chat platform. Provides chat rooms for all things Sponge. Sponge's IRC server is synced with Sponge's Discord server.
Isu
Sebuah cara untuk menandai masalah, bug, perbaikan, dan hal-hal yang harus dilakukan pada GitHub dan tempat lain. Proyek Sepon menggunakan GitHub sebagai Pelacak Masalah.
Java
Bahasa pemrograman yang Minecraft dan Sepon ditulis dalam, dan perlu dijalankan. https://java.com/en/
MCP
Mod Paket Coder membantu pengembang membuat mods untuk server dan klien Minecraft. http://www.modcoderpack.com/website
Mixin
Specialized program components that inject Sponge into Minecraft. https://github.com/SpongePowered/Mixin/wiki
Mod
Modifikasi Minecraft yang mengubah gameplay entah bagaimana. Modifikasi yang ditulis menggunakan Forge API membutuhkan Forge untuk bekerja, namun beberapa mods dapat terinstal sendiri.
NPC
Non-Player Character. Any character not controlled by a player, e.g. a Villager.
Bijih
The Official Sponge plugin hosting site. https://ore.spongepowered.org/
Patreon
Layanan yang memungkinkan sumbangan untuk proyek dan organisasi. Jika Anda ingin mendukung Sepon, ini adalah tempat untuk melihat! Sumbangan Sepon Patreon <https://www.patreon.com/Sponge>
Plugin
A Minecraft mod that extends and changes Minecraft via SpongeAPI, usable only on Sponge servers.
Pemimpin Proyek
The folks in charge of the entire Sponge Project, responsible for coordinating the activities of the various Teams and sub-projects.
Permintaan Tarik (PR)
Sebuah metode untuk mengirimkan kontribusi ke proyek pengembangan terbuka, seperti repositori Sepon di GitHub.
Sepon
Nama keseluruhan proyek.
SpongeAPI
API Plugin Minecraft dikembangkan oleh proyek Sepon.
SpongeAuth
The authentication panel and SSO solution for all of the Sponge websites. https://auth.spongepowered.org/
Sepon umum
Sebuah repositori kode umum untuk implementasi SpongeAPI resmi, digunakan dalam membangun mereka. SpongeVanilla dan SpongeForge menggunakan repositori ini.
SpongeForge
Implementasi SpongeAPI sebagai Forge Coremod.
Dokumen Sepon
Didedikasikan dan sering diperbarui dokumentasi, sumber informasi terbaik pada Sepon.
Forum Sepon
Home to the Sponge Project and the second-best source of information on Sponge. Update announcements are posted regularly. https://forums.spongepowered.org/
Dasar Sepon
Sisi keuangan proyek, entitas terpisah yang menerima, mengelola dan mendistribusikan sumbangan.
SpongeVanilla
An implementation of SpongeAPI that runs on Vanilla Minecraft (and not Forge).
Spongie
Maskot Sepon, Sepon kecil yang gagah berani dengan jubah merah.
Spongie
Sepon GitHub Bot yang membuat PR pratinjau untuk Dokumen.
Tim Pemimpin
The people who direct the Teams (WebDev, Docs, SysOps, subreddit, IRC).
