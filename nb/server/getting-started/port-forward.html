Portvideresending¶

Hvis du kjører Sponge-serveren din hjemmefra, er det nødvendig å sette opp portvideresending («Port Forwarding») på ditt modem eller din ruter for at andre skal kunne koble til.

Advarsel

Pass på at du tar nødvendige forhåndsregler når du videresender porter, fordi det kan være usikkert.

Minecraft, og dermed Sponge, bruker i utgangspunktet port 25565. Derfor må port 25565 videresendes til den interne IP-en til din datamaskin. UDP og TCP er protokollene som må videresendes.

Tips

Du kan endre på hvilken port serveren bruker ved å endre på den riktige nøkkelen i din server.properties.

Portvideresending kan settes opp gjennom administrasjonspanelet til din router. Hvis du ikke vet hvordan du kobler til din router sitt administrasjonspanel, bør du søke på internett etter spesifikke instruksjoner for å sette opp portvideresending i din router. Instruksjonene er vanligvis forskjellige fra router til router.

Hvis du foretrekker at serveren din skal være online idet du starter maskinen, kan det hende du må videresende porter via maskinvare.
