Lage et startskript¶

Obs

Disse instruksjonene gjelder bare hvis du planlegger å kjøre Minecraftserveren din fra din egen maskin. De fleste felles Minecraft-vertene lager et startskript for deg.

Skrive et startskript¶

Først, åpne et tekstredigeringsprogram slik som Atom, Sublime Text, eller Notisblokk. Skriv (eller lim inn) et startskript til serveren din. Eksempler på enkle startskript til Windows, macOS og Linux er vedlagt under. Pass på at du ikke bruker mer RAM enn maskinen din har.

Obs

De følgende eksemplene er generiske. For en Forge-server med SpongeForge, endre forge-1.12.2-XYZ-universal.jar til hva enn Forge-versjonen i mappen til din server heter. For å starte en SpongeVanilla-server, endre forge-1.12.2-XYZ-universal.jar til navnet på SpongeVanilla.jar-filen.

Windows¶

java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar
pause

Lagre ditt Windows-startskript som start.bat.

Advarsel

På grunn av Quick Edit Mode som brukes i Ledetekst og PowerShell, kan det skje at serveren din henger seg opp. Denne modusen fryser prosessen når du markerer noe eller trykker i konsollvinduet. I løpet av denne perioden vil du få beskjed i loggen om at serveren hoppet over et antall ticks som kan tilsvare mange sekunder eller minutter. Du kan unngå dette ved å aldri markere tekst eller trykke med musepekeren i vinduet, eller ved å skru av Quick Edit Mode i Egenskaper-dialogen.

macOS¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Lagre ditt Mac-startskript som start.command.

Linux¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.12.2-XYZ-universal.jar

Lagre ditt Linux-startskript som start.sh.

Kjøre et startskript¶

Pass på at du kjører startskriptet ditt fra en mappe som er laget spesifikt for din server. Dette er for din egen skyld; dessverre kan ikke Spongie suge til seg tårene dine hvis du ikke gjør dette.

Du kan kjøre startskriptet ditt ved å dobbeltklikke det. Hvis du bruker en konsoll eller terminal, naviger deg til mappen der skriptet ligger og kjør det. Husk at du må godta Mojang sin sluttbrukeravtale for å kjøre en server.

Obs

Det innebygde grafiske grensesnittet til Minecraft-serveren deaktiveres av Sponge, fordi den bruker store mengder prosessorkraft.

Advarsel

Hvis du får en feilmelding om tilganger når du prøver å starte serveren din på en Mac, prøv dette:

  • Åpne Terminal.
  • Skriv chmod a+x med et mellomrom på slutten.
  • Dra startskriptet ditt til Terminalvinduet.
  • Trykk på enter-tasten.
