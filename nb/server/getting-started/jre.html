Installere Java¶

Java er påkrevd for at du skal kunne kjøre Sponge og Minecraft: Java Edition. Mest sannsynlig har du allerede Java installert, men det kan hende at du må oppdatere til en nyere versjon.

Sponge krever for tiden Java 8 (spesifikt 1.8.0_20 eller høyere). Eldre versjoner av Java er utdatert, og vil ikke fungere med Sponge. Forskjellen mellom hovedversjoner av Java (6, 7, 8) er stor, og eldre versjoner kan ikke kjøre Sponge på en ordentlig måte. For tiden fungerer Sponge heller ikke på Javaversjoner som er større enn 8.

Installere Java¶

Hvis du har Windows eller macOS på din maskin, kan du laste ned Java fra den offisielle nettsiden.

Linux-brukere kan installere OpenJDK via sin pakkehåndterer. OpenJDK er en versjon av Oracle sin Java med åpen kildekode, og bør fungere minst like bra. Det er imidlertid også mulig å laste ned Oracle sin versjon for Linux, men du bør være klar over at mange Linuxpakker som krever Java vil installere OpenJDK likevel.

Det kan hende du må konfigurere pekeren til din JRE/JDK i din

  • launcher med Profil (Avanserte innstillinger) -> <Din profil> -> Java-Executable (.../java.exe)
  • server, enten ved å sette PATH-variablen, eller ved å bruke hele stien til din Javafil

32-bit eller 64-bit¶

Hvis datamaskinen din støtter det, bør du alltid bruke 64-bit-versjoner av Java. Installasjonsprogrammene fra denne nettsiden bør merke om din datamaskin støtter 64-bit. Det hender imidlertid at automatikken tar feil (prøv en annen nettleser om du er usikker).

Det er også mulig å finne ut av dette i systeminformasjonsvinduet.

  • Windows: Trykk <Win> + <Pause>, eller se på din systemtype gjennom System i Kontrollpanel.
  • Linux: åpne en terminal og skriv arch eller uname -m. i686 betyr at du har 32-bit, og x86-64 betyr at du har 64-bit.
  • Mac: det finnes kun én nedlasting uansett…

Hvis du er sikker på hvilken arkitektur din maskin bruker, kan du velge riktig installasjonsprogram selv.

Fordi 64-bit-versjonen av Java kjører mye bedre, og også lar Java bruke mer enn ca. 3 GB med RAM (minne), så anbefaler vi alltid 64-bit over 32-bit.

De fleste moderne datamaskiner støtter 64-bit.

JDK eller JRE¶

JRE (Java Runtime Environment, Javakjøretidsmiljø) brukes for å kjøre Javaprogrammer. Nedlastingssiden som er lenket til lenger oppe tilbyr JRE.

JDK (Java Development Kit, Javautviklersettet) brukes til å lage Javaprogrammer, og du trenger det ikke med mindre du har tenkt til å lage Spongeplugins eller jobbe på Sponge. I noen tilfeller kan det imidlertid hende du trenger JDK for å diagnostisere et kjørende Javaprogram, slik som Sponge. Du kan laste ned JDK fra en annen nettside.
