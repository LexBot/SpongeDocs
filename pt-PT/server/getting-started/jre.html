Instalar o Java¶

O Java é necessário para correres o Sponge e o Minecraft: Java Edition. Provavelmente já tens o Java, mas poderás ter de o atualizar.

O Sponge requer o Java 8 (especificamente, a versão 1.8.0_20 ou superior). As versões mais antigas são de utilização não recomendada e não funcionarão com o Sponge. A diferença entre as principais versões do Java (6, 7, 8) é bastante significativa, e as versões antigas não conseguem executar o Sponge corretamente. Neste momento, o Sponge não corre com versões de Java superiores à 8.

Instalar o Java¶

Se tens o Windows ou o macOS no teu computador, podes descarregar o Java do site oficial.

Os utilizadores do Linux podem instalar o OpenJDK através do package manager que usarem. O OpenJDK é a versão open-source do Java da Oracle, e deve fncionar de igual forma, se não melhor. No entanto, também é possível descarregar a versão oficial da versão da Oracle para Linux, mas tem em mente que muitos pacotes que dependem do Java instalam o OpenJDK quando são instalados.

Podes ter de configurar o caminho para o JRE/JDK no teu:

  • launcher, utilizando Profile (Advanced Options) -> <o teu perfil> -> Java-Executable (.../java.exe)
  • servidor, ou usando a variável PATH, ou usando o caminho absoluto para o teu executável Java

32-bit vs. 64-bit¶

Se o teu computador suportar 64 bits, deves utilizá-lo sempre que possível. Os instaladores do Java deste website devem detetar automaticamente se o teu computador suporta 64 bits. No entanto, a deteção falha, por vezes (utiliza diferentes browsers se estiveres incerto).

Também é possível retirar esta informação da janela de informações do sistema.

  • Windows: Prime``<Win>`` + <Pause> ou via Systeminformation vê o teu system type.
  • Linux: Abre um terminal e escreve arch ou uname -m. i686 -> 32-bit. x86_64 -> 64-bit.
  • Mac: Só há uma opção de download…

Se sabes qual é a versão do teu sistema operativo, podes escolher manualmente o instalador correto.

Recomendamos sempre o Java de 64 bits pois corre bastante melhor do que o Java de 32 bits e permite alocar mais de ~3 GB de RAM (memória).

A maior parte dos computadores modernos suporta 64 bits.

JDK vs. JRE¶

O JRE (Java Runtime Environment) é utilizado para executar aplicações do Java. A página de download do link acima dá acesso ao JRE.

O JDK (Java Development Kit) é usado para criar aplicações em Java, e não vais precisar dele a menos que planeies fazer plugins para o Sponge ou trabalhar no Sponge. No entanto, em alguns casos, poderás preisar do JDK para diagnosticar uma aplicação Java, como o Sponge. Podes descarregar o JDK de um site diferente.
