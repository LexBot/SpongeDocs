Instalar o SpongeForge¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it’s more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Os utilizadores que não quiserem usar o Minecraft Forge podem considerar o SpongeVanilla.

Download¶

Review our Versioning and grab your copy of SpongeForge here.

Instalar o SpongeForge¶

Nota

Se utilizas (ou planeias vir a utilizar) um host de servidores de jogos, eles poderão ter um painel que instale o Sponge por ti.

Nota

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_74. Note that Sponge requires at least 1.8.0_20 or above to run properly but it is recommended to use the latest Java 8 version.

Um Jogador / Servidor LAN In-Game¶

  1. Descarrega o instalador do Minecraft Forge do site do Minecraft Forge. Certifica-te de que usas exatamente a build com o mesmo número que a mostrada acima.
  2. Executa o instalador do Forge fornecido. Um novo perfil Forge será criado no launcher do Minecraft.
  3. Abre o laucher do Minecraft e seleciona o novo perfil do Forge que foi criado.
  4. Clica em «Options» e em «Open Game Dir».
  5. Descarrega o SpongeForge do site do Sponge e coloca-o dentro da pasta «mods». Se a pasta não existir, cria-a.
  6. O Sponge deve funcionar tanto em Single Player como num mundo aberto para a LAN.

A seguir, aprende como podes configurar o Sponge e como podes gerir o teu servidor (incluindo a instalação de plugins).

Servidores Dedicados¶

Nota

Se já tens um servidor Forge, basta pores o Sponge mod na pasta mods. Lembra-te de atualizares a versão do Forge para a versão específica que o SpongeForge requer. Dá uma olhadela ao topo desta página se não sabes que versão utilizar.

Instalar o Forge pela linha de comandos¶

  1. Visita o site do Minecraft Forge <https://files.minecraftforge.net/>`e clica em «Show all downloads» para veres uma lista completa das opções disponíveis. Identifica a versão que corresponde à indicada :doc:`no nome do ficheiro do SpongeForge que descarregaste <spongeforge>, e coloca o rato por cima do (i) ao lado de «installer» para teres o link de download direto.
  2. Usa o teu método de download preferido para descarregares o Jar para o seu destino. Por exemplo: wget http://url.to/forge-version-installer.jar
  3. Na pasta onde desejas instalar o Forge, executa o Jar com a opção --installServer. Exemplo: java -jar forge-version-installer.jar --installServer
  4. Salat para a secção Ligar o SpongeForge ao Forge, abaixo.

Instalar o Forge por uma Interface Gráfica¶

  1. Descarrega o instalador do Minecraft Forge do `site do Minecraft Forge <https://files.minecraftforge.net/>`que corresponde à versão mostrada no nome do ficheiro do SpongeForge que descarregaste. Vê o esquema de nomes do SpongeForge e do Forge acima.
  2. Corre o instalador do Forge fornecidom selecionar «Instalar Servidor», escolhe uma pasta vazia para colocar os ficheiros do servidor e depois clica em OK.
  3. Salat para a secção Ligar o SpongeForge ao Forge, abaixo.

Ligar o SpongeForge ao Forge¶

  1. Descarrega o SpongeForge do site do Sponge e coloca-o dentro da pasta «mods» na pasta do teu servidor. Se a pasta não existir, cria-a.
  2. Agora podes correr o servidor via comando ou launch script java -jar forge-version-XYZ.jar.
  3. Se estiveres a trabalhar a partir de casa, configura o Encaminhamento de Portas para que os outros se possam juntar.

De seguida, aprende a criar e utilizar um launch-script, configurar o Sponge e gerir o teu servidor (incluindo a instalar plugins).
