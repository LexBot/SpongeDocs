Resolução de Problemas¶

Provavelmente estás aqui porque alguma coisa correu mal no teu servidor Sponge. Vamos ver se conseguimos descobrir o problema, e como o resolver.

Potential Sources of Trouble

  • O Java não está Instalado no teu Computador
  • Falha na Ligação de Rede (ou Ataque DDoS)
  • Não tem Memória Livre Suficiente
  • Malformed Config File (e.g. Bad Editing)
  • Um Plugin (ou um Mod) Está Avariado
  • Operating System Unstable (e.g. Virus Infection)
  • Dados Corrompidos
  • Problema entre o Teclado e a Cadeira
  • Há um Bug no Sponge
  • Há Algo Errado no Universo

O Java não está Instalado no teu Computador¶

Solution: Get Java. Visit the Instalar o Java for more information.

Falha na Ligação de Rede (ou Ataque DDoS)¶

Sintomas: A ligação de rede está muito atrasada, liga e desliga, ou é inexistente.

Soluções: Verifica a tua ligação ao modem/router. Vê se o teu browser tem problemas semelhantes. Podes utilizar um serviço gratuito, como o speedtest.net, para verificar a velocidade da tua ligação. Outros serviços a correr no teu computador ou na sua rede podem ser a causa. Certifica-te de que tens o reencaminhamento de portas configurado no teu router. Um ataque DDos deverá interromper a tua ligação à Internet completamente, e é importante que contactes o teu ISP se achares que é o caso.

Não tem Memória Livre Suficiente¶

Sintomas: O Servidor para, geralmente acompanhado de uma mensagem do género de «Out of Memory».

Solutions: Expand the maximum Perm memory size with the startup argument -XX:MaxPermSize=128. Expand your server heap memory (if possible) with startup arguments e.g. -Xms1024M (1GB starting memory) and -Xmx2048M (2GB maximum). Monitor your free memory on the computer and see if there is some locked up in other processes. You may need to kill frozen java processes, or restart your machine. Memory leaks sometimes occur with bugs in plugins, which can take time to isolate.

Ainda tens problemas?: Se ainda estiveres a ter problemas apesar do descrito acima e não conseguires aumentar a memória Heap, verifica no gestor de tarefas se estás a utilizar toda a memória disponível. Se estiveres, a única solução é adicionares mais RAM ao teu sistema. Se houver muita memória disponível, estás a utilizar o Java de 32 bits. Se estiveres a usar o JAva de 32 bits, recomendamos que atualizes o Java para a versão de 64 bits, desde que o teu sistema operativo também seja de 64 bits.

Malformed Config File (e.g. Bad Editing)¶

Sintoma: Um (ou mais) plugins recusam-se a iniciar, oiu apresentam um comportamento estranho. Os ficheiros de registo (logs) do servidor podem conter mensagens acerca de erros ao ler ficheiros no arranque. O servidor para, e os dados estão corrompidos.

Solução: Para o servidor e verifica os ficheiros que editaste. Substitui os ficheiros corrompidos por cópias de segurança. Poderás precisar de apagar completamente a configuração para que o servidor a volte a gerar no arranque.

Um Plugin (ou um Mod) Está Avariado¶

Sintoma: Pode ser quase qualquer coisa - o que quer o plugin faça mais o tal fator. Geralmente, o servidor para, escrevendo uma série de erros nos ficheiros de registo.

Solution: Stop the server, and check to see nothing has been corrupted. Be sure to check that it isn’t from an incorrectly edited config file (above). Remove suspect plugins and add them again one by one, restarting the server each time. The problem may originate from one plugin that is out of date - check for updates. Plugin conflict may also be the cause, having two incompatible plugins. You can also check our Mod Compatibility page to see if the problem is already known, and if there’s a way to fix it.

Operating System Unstable (e.g. Virus Infection)¶

Sintoma: O servidor está constantemente a parar, e outras partes do seu sistema operativo também apresentam problemas.

Solution: Stop everything. Thoroughly check your system and storage devices for malware and viruses. Good tools for this include AdwCleaner, Junkware Removal Tool, MalwareBytes, and most antivirus ware. Check your server files for corruption after a clean restart of your system. Examine the hardware for damage too if the problems persist - e.g. a faulty power supply.

Dados Corrompidos¶

Sintoma: Os ficheiros do mundo não carregam ou o servidor para quando os jogadores entram em certas chunks. A base de dados está corrompida.

Solução: Substitua os ficheiros corrompidos por uma cópia de segurança. Há software para reparar mundos corrompidos, e as regiões em falta podem ser geradas novamente. Investigue a causa do problema - era um plugin defeituoso? Um driver de base de dados avariado? Uma falha de energia? Ou outra coisa? Certifique-se de que faz cópias de segurança de dados importantes para um dispositivo seguro regularmente the cause of corruption - was it a malformed plugin, database driver, power failure or something else? Always make sure you make regular backups of important data onto a secure device.

Problema entre o Teclado e a Cadeira¶

Sintoma: Estava tudo a funcionar bem ontem, hoje fiz XYZ e ficou tudo estranho…

Solução: O SpongeDocs não é suficientemente grande para abordar todos os problemas possíveis. Vale sempre a pena pensar no que fez recentemente que poderá ter afetado o servidor. Uma placa de memória pode estar solta depois de uma limpeza, um atalho pode ter deixado de funcionar, etc.

Há um Bug no Sponge¶

Sintoma: Nenhum dos enunciados acima e, ainda assim, o servidor não funciona como esperado.

Solução: Para grandes males, grandes remédios! Preenche um relatório no Issue Tracker do`SpongeForge <https://github.com/spongepowered/SpongeForge/issues>`_ ou do`SpongeVanilla <https://github.com/spongepowered/SpongeVanilla/issues>`_, incluindo detalhes sobre a versão que estás a utilizar, e um link para ficheiros de registo relevantes.

Há Algo Errado no Universo¶

Não podemos ajudá-lo neste aspeto. Desenrasque-se.
