Ore Web API¶

O Ore oferece uma API JSON RESTful para interagir com os mods e plugins indexados pelo Ore. Seguem-se as especificações da referida API.

Tipos de Dados¶

Existem alguns tipos de dados baseado em inteiros, do tipo enum, que são utilizados por toda a API. Segue-se uma lista desses tipos de dados.

IDs de Categoria:¶

0 Ferramentas de administração
1 Ferramentas de chat
2 Ferramentas de Desenvolvedor
3 Economia
4 Modo de Jogo
5 Jogos
6 Proteção
7 Role Playing
8 Gestão do mundo
9 Diversos
10 Indefinido

IDs de métodos de ordenação:¶

0 Mais estrelas
1 Mais descarregados
2 Mais vizualizações
3 Mais recentes
4 Atualizados recentemente

Rotas¶

Abaixo está uma lista das rotas que estão atualmente disponíveis dentro da API.

  • Listar Projetos
  • Obter Projeto
  • Listar Versões de um Projeto
  • Obter a Versão do Projeto
  • Enviar uma versão do projeto
  • Listar utilizadores
  • Obter utilizador
  • Transferir uma versão do projeto

Rotas descontinuadas¶

As seguintes rotas foram descontinuadas e não devem ser usadas. As rotas novas, que substituem estas, serão provavelmente diferentes destas, pelo que deves consultar a documentação antes de migrares.

Descontinuada Substituição
/api/projects /api/v1/projects
/api/users /api/v1/users
/api/projects/:pluginId/version /api/v1/projects/:pluginId/version
/api/projects/:pluginId/versions/:version /api/v1/projects/:pluginId/versions/:version
/api/projects/:pluginId /api/v1/projects/:pluginId
/api/users/:username /api/v1/users/:username
/api/projects/:pluginId/versions/:name /api/v1/projects/:pluginId/versions/:name
