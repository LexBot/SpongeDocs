Ativos Artísticos¶

Esta página fornece o logótipo oficial do SpongePowered e a mascote. A fonte usada no logo é Montserrat. Estás à vontade para utilizá-los para espalhar a palavra sobre o Sponge. No entanto, tem em consideração que estas imagens não são licenciadas sob a licença MIT, mas sob a licença Creative Commons - Non Commercial, No Derivatives license. Vê as informações abaixo para perceberes a nossa intenção.

Nota

Esta é uma versão traduzida. Por favor, tem em conta que é a licença em inglês que conta. As versões traduzidas só servem para fins informativos.

Podes:

  • Fazer pequenas alterações às expressões faciaIs da Spongie - tais como fazer um sorriso, fazê-la perplexa, ou triste, adicionar um chapéu, etc. Deverá ser fácil perceber que o Spongie que criaste é a mascote Spongie, mas a vestir uma roupa diferente ou com uma atitude diferente.
  • Usar o Spongie ou o logótipo do SpongePowered num artigo ou numa publicação num blog sobre o Projeto Sponge para espalhar a palavra.
  • Usar o logótipo do SpongePowered num link para a Sponge Homepage (por exemplo na tua assinatura num fórum etc.) ou noutros recursos oficiais do SpongePowered.
  • Usar os botões nas redes sociais.

Não podes:

  • Usar a mascote como link para o Projeto Sponge somente. O logótipo do SpongePowered (que retém a forma do Sponge), é uma melhor representação do Projeto Sponge quando utilizado como um afiliado ou referência.
  • Criar uma versão muito modificada da Spongie, em que se torne irreconhecível ou claramente diferente da mascote Spongie original.
  • Adicionar a imagem do teu projeto à Spongie, ou vice-versa.
  • Afirmar que a Spongie é a tua própria mascote, ou usá-la como uma mascote para o teu próprio projeto.
  • Vender ou usar a Spongie ou o seu logo em atividades comerciais sem permissão.
  • Alterar quaisquer cores ou dimensões.
