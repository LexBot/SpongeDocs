Instalar o Git¶

O Git é um sistema de controlo de versão, de código-fonte aberto, que ajuda muito o desenvolvimento de projetos colaborativos.

O projeto Sponge, bem como milhares de outros projetos de código-fonte aberto, hospeda os seus repositórios Git no GitHub. Assim, o Git é uma ferramenta crucial no desenvolvimento do Sponge e da SpongeAPI.

O site do Git tem bastante documentação, e a página de downloads deles tem bastantes clientes com interface gráfica para vários sistemas operativos.

Download¶

Windows¶

O GitHub Desktop é um método fácil de instalar o Git no Windows, uma vez que o Git vem incluído no software.

É recomendado que reinicies o teu computador depois de instalares o Git.

Mac¶

Existem algumas maneiras de instalar o Git no macOS.

O método mais fácil é instalar as Xcode Command Line Tools.

Aviso

Estas instruções não funcionam em Macs que estejam a utilizar uma versão anterior ao Mavericks. Se estiveres a usar uma versão do OS X mais antiga que o Mavericks, instala o cliente do GitHub em vez disto.

  1. Abre o Terminal
  2. Executa xcode-select --install.
  3. Instala-o, e come uma bolachita enquanto esperas.
  4. Executa git no Terminal.

Como alternativa, podes instalar o GitHub Desktop. O Git faz parte da instalação do GitHub Desktop.

É recomendado que reinicies o teu computador depois de instalares o Git.

Linux e Unix¶

O método mais simples de instalar o Git no Linux é usando o package manager que veio com tua distribuição de Linux.

Nota

Poderás ter de prefixar os comandos abaixo com sudo.

  1. Abre o Terminal
  2. Executa apt-get install git se estiveres a usar o Ubuntu ou o Debian. Executa yum install git se estiveres a usar o Fedora.

O GitHub Desktop não está disponível para Linux de momento, só há para Windows e Mac.

É recomendado que reinicies o teu computador depois de instalares o Git.

Configuração¶

Quem És?¶

Antes de começares a trabalhar com o GIt e com o repositório, certifica-te de que a tua configuração do GIT tema tua identidade. Abre a linha de comandos e escreve:

git config --list

Procura as strings user.name e user.email. Se não estiverem definidas como o mesmo username e email da tua conta do GitHub, então define-os:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Dica

Não faças nenhuma contribuição em nenhum repositório do Sponge sem definires o teu user.name e user.email.
