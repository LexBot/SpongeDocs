서버 만들기¶

이 섹션의 문서들은 SpongeAPI 의 구현물을 통해 서버를 만들고 실행하는 방법에 대해 설명합니다.

Sponge의 구현물을 이용해 서버를 만들거나 이전하고자 하신다면 여기를 읽어주세요. 플러그인 개발자분들에게도 도움이 될 수 있습니다, 플러그인 개발 중 테스트 서버가 필요할 수도 있으니까요.

주석

우리는 가능하면 implementation-agnostic (기본적으로 하나의 SpongeAPI 구현물에 치우치지 않는 것) 가 되도록 노력할 것입니다. 그렇더라도, 실수는 언제든지 일어납니다.
