Java 설치¶

Java는 Sponge와 Minecraft: Java Edition을 실행하는 데 필요합니다. 당신은 이미 Java를 가지고 있을 가능성이 높지만, 업데이트가 필요할 수도 있습니다.

Sponge는 현재 Java 8 (1.8.0_20 이상) 을 요구합니다. 이 보다 오래된 버전은 Sponge와 호환되지 않습니다. Java 6, 7, 8처럼 주요 버전의 차이는 확연합니다, 그리고 구버전에서 Sponge를 올바르게 동작할 수 없습니다. 현재 Sponge는 Java 9 버전에서 동작하지 않을 것입니다.

Java 설치¶

당신의 컴퓨터에서 Windows나 macOS를 사용하고 계신 경우, 공식 사이트에서 다운로드 하실 수 있습니다.

Linux users can install OpenJDK via their package manager. OpenJDK is the open source version of the Oracle version of Java, and it should work just as well, if not better. However, it’s also possible to download the Oracle version for Linux, but be aware that many Java-dependent Linux packages will still install OpenJDK anyway.

JRE/JDK 경로를 지정해야 할 수도 있습니다:

  • 게임 런처에서 실행 설정 (고급 설정 On) -> <Your Profile> -> Java 실행 파일 (.../java.exe)
  • Sponge 서버를 돌리려면 운영 체제에 Java PATH 변수를 설정하거나 실행 스크립트에 Java 파일 경로를 제공해야 합니다.

32비트 vs 64비트¶

If your computer supports it, you should use 64-bit versions of Java whenever possible. The Java installers from the this website should detect whether your computer is ready for 64-bit. However, the autodetection is sometimes wrong (try different browsers, if you are unsure).

당신이 사용하는 운영 체제에 맞게 시스템 정보를 확인할 수 있습니다.

  • Windows: Press <Win> + <Pause> or via Systeminformation have a look at your system type.
  • 리눅스: 터미널을 열고 arch 또는 uname -m 을 입력하세요. 32-bit이면 i686, 64-bit이면 x86_64 로 출력됩니다.
  • Mac: Mac 운영 체제는 무조건 64-bit 입니다.

당신의 시스템 아키텍처가 무엇인지 확실하다면 수동으로 적절한 설치 프로그램을 찾아서 Java를 설치할 수 있습니다.

64비트 Java는 생각만큼 잘 동작하고, 3GB 이상의 램(메모리) 을 할당시킬 수 있기 때문에, 32비트 사용은 권장되지 않습니다.

요즘 대부분의 컴퓨터들은 64비트를 지원합니다.

JDK와 JRE¶

JRE (Java Runtime Environment)는 Java 애플리케이션을 실행하는 데 사용됩니다. 위에 제공된 링크는 JRE를 제공합니다.

JDK (Java Development Kit)은 Java 애플리케이션을 만드는 데 사용됩니다. Sponge에 기여하거나, Sponge 플러그인을 만들 계획이 없다면 필요하지 않을 겁니다. 하지만 몇몇 경우에서는, Sponge 같은 Java 애플리케이션의 문제를 진단하기 위해 JDK가 필요할 수 있습니다. JDK는 다른 사이트에서 받을 수 있습니다.
