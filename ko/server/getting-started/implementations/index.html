구현 프로그램 선택하기¶

구현 프로그램은 Sponge 플러그인을 작동시켜 줍니다. 당신의 플러그인이 SpongeAPI를 기반으로 만들어졌다면, 아래에 설명할 구현물 중 어느 것에서나 동작할 것입니다.

구현 프로그램 없이는 마인크래프트에서 Sponge 플러그인을 동작시킬 수 없습니다.

SpongeAPI는 `오픈 소스<https://github.com/SpongePowered/SpongeAPI>`__ 프로젝트입니다.

사용할 수 있는 구현 프로그램¶

구현 프로그램은 현재 두 가지가 존재합니다:

이름 기반 프로그램
SpongeForge 《순정》 Minecraft: Java Edition, Forge
SpongeVanilla 《순정》 Minecraft: Java Edition

어떤 것을 선택해야 하나요?¶

서버에서 Forge 모드를 사용하거나, 싱글 플레이에서 Sponge를 쓰려는 경우 SpongeForge 를 선택하세요.

만약 (모드는 없고) 플러그인을 사용할 수 있는 마인크래프트 서버를 열고 싶은 경우, SpongeForge 와 SpongeVanilla 중에서 선택하세요. SpongeForge를 선택했다면, 서버에 모드를 추가하지 않는다는 가정 하에 서버에 접속할 때 클라이언트 모드가 요구되지 않습니다. Forge 없는 순정 마인크래프트 서버를 원하신다면, SpongeVanilla를 사용하는 것이 적합합니다.

SpongeVanilla와 (모드가 추가되지 않은) SpongeForge 서버는 똑같이 동작합니다. 따라서 두 구현물 중 하나를 선택하는 것은 자신이 구상하는 서버, 즉 여러분의 취향에 달렸습니다.
