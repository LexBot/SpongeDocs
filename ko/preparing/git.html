Git 설치하기¶

Git은 오픈소스 버전 제어 시스템으로, 공동 프로젝트 개발에 엄청난 도움을 줍니다.

The Sponge project, alongside thousands of other open-source projects, hosts its Git repositories on GitHub. Thus, Git is a crucial tool in the development of Sponge and SpongeAPI.

Git 웹사이트 에 상당한 설명 문서가 있고, 다운로드 페이지에서 여러 가지 운영 체제에 맞는 다양한 GUI 클라이언트를 선택할 수 있습니다.

내려받기¶

Windows¶

GitHub Desktop 으로 Windows에 Git을 쉽게 설치할 수 있습니다. Git이 소프트웨어에 포함되어 있습니다.

Git을 설치한 뒤 컴퓨터를 재시작하는 것을 권장합니다.

Mac¶

macOS에 Git을 설치하는 방법은 여러 가지가 있습니다.

가장 쉬운 방법은 Xcode 명령 줄 도구를 설치하는 것입니다.

경고

이 지침은 Mavericks보다 오래된 버전을 구동하는 Mac에서는 작동하지 않습니다. Mavericks보다 오래된 버전의 OS X를 구동하고 있다면, 대신 GitHub 클라이언트를 설치하십시오.

  1. 터미널을 시작하십시오.
  2. xcode-select --install 을 실행하십시오.
  3. 설치하고, 기다리는 동안 쿠키를 먹으십시오.
  4. 터미널에서 ``git``을 실행하십시오.

그 대신에, ``GitHub Desktop <https://desktop.github.com/>`_ 을 설치해도 됩니다. Git은 GitHub Desktop 설치의 일부로 제공됩니다.

Git을 설치한 뒤 컴퓨터를 재시작하는 것을 권장합니다.

Linux와 Unix¶

Linux에 Git을 설치하는 가장 간단한 방법은 Linux 배포판에 포함된 패키지 관리자를 사용하는 것입니다.

주석

명령어들 앞에 sudo 를 붙여야 할 수도 있습니다.

  1. 터미널을 시작하십시오.
  2. Run apt-get install git if you are on an Ubuntu or Debian-based distribution. Run yum install git if you are on Fedora.

Windows와 Mac과는 달리, Linux에서는 현재 GitHub Desktop을 사용할 수 없습니다.

Git을 설치한 뒤 컴퓨터를 재시작하는 것을 권장합니다.

Setup¶

당신은 누구입니까?¶

git과 저장소로 작업을 시작하기 전에, git 설정에 당신의 신원을 설정해야 합니다. CLI를 열고 다음을 입력하십시오.

git config --list

user.name 과 user.email 을 찾으십시오. GitHub 계정과 같은 사용자명과 이메일이 아니라면 설정하십시오.

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

참고

user.name 과 user.email 을 설정하기 전에는 어떠한 Sponge 저장소에 어떠한 기여도 하지 마십시오.
