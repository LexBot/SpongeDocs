작업 공간 준비하기¶

이 부분은 당신이 Sponge 플러그인 개발을 준비하도록 기초적인 IDE 설정을 하는 방법을 보여줍니다.

현재, 우리는 두 개의 가장 인기있는 Java IDE 인 IntelliJ 와 Eclipse 에 대한 설명문을 제공합니다. 하지만 Sponge 플러그인을 개발하는 데에는 다른 많은 방법들이 존재합니다. 몇몇은 Netbeans 를 사용하는 것을 선택하거나 간단하게 VS Code, Atom, 그리고 Sublime Text 와 같은 텍스트 에디터를 사용하는 것을 선택합니다.
