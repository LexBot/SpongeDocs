参与到 Sponge 的开发工作来¶

这部分描述了当你打算开发 Sponge 本身时，了解项目的最好方法。在试图参与开发前，你必须熟悉 git 和我们的 代码规范 中的内容。 Sponge 项目的架构 中提到的基本知识也很帮助。

如果你想加入我们 Sponge 项目，请直接转到 提交拉取请求 页面。
