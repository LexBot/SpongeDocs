未来计划¶

我们要做的工作还有很多，尤其是：

  • 如果 Mojang 推出正式 Mod API，我们会考虑在这个 API 上实现 Sponge API ，这样基于 Sponge 的插件就可以同时运行在两种框架之上了。
  • 为了让客户端的功能更加多样化，我们或许会考虑一些仅在客户端运行的 Sponge 模组。

同样的，我们也有些不会考虑的事情：

  • 我们不会原生支持 Bukkit 插件。
  • 出于安全考虑，我们不会考虑从服务端向直接发送 Mod 。
  • 我们的工作重心在 Java 上，也不会使用其他的编程语言来完成 Sponge 的实现，但我们仍然鼓励其他人在这上面进行尝试。
