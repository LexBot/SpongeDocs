设置开发环境¶

这一部分将向你展示如何设置你的 IDE，以完成开发 Sponge 插件的基本设置。

我们目前只提供两种主流 Java IDE 的开发环境配置文档：Intellij 和 Eclipse，但实际上开发 Sponge 插件完全可以使用其他的工具，比如 Netbeans 或者诸如 VS Code、Atom、Sublime Text 这样的文本编辑器。
