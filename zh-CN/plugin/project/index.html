设置你的项目¶

在你开始开发你的 Sponge 插件之前，你需要在你的项目中设置 SpongeAPI 的依赖。如果你有使用构建系统的经验，下面是你需要添加的 Maven 仓库。如果没有，这里分别是 Gradle 和 Maven 的更详细说明。

SpongeAPI 依赖¶

Maven 仓库¶

名称 sponge
URL https://repo.spongepowered.org/maven

Maven 依赖¶

Group ID org.spongepowered
Artifact ID spongeapi
Version 例如：7.2.0（稳定版），或者 8.0.0-SNAPSHOT（开发版，不稳定）
