安装 SpongeVanilla¶

SpongeVanilla 是建立在独立的原版 Minecraft 服务端上的 SpongeAPI 实现。

综述¶

SpongeVanilla 是以 Patch 的方式维护的建立在原版 Minecraft 服务端上的 Sponge API 实现。这意味着它是一个不依赖于 Minecraft Forge 或者 Forge Mod Loader（FML）的独立的 Sponge API 实现。SpongeVanilla 的开发工作与 Forge 版本的 Sponge 同时进行，用于使不希望使用 Forge 服务端的用户可以不依赖 Forge 运行 Sponge 服务器。SpongeVanilla 的前身是一个名为 Granite 的 AzureusNation 和 VoltaSalt 共同开发的独立服务端，不过已于 2015 年三月合并进 Sponge 开发团队。

相关下载¶

Review our 版本规范 and grab your copy of SpongeVanilla here.

安装 SpongeVanilla¶

注解

如果你使用（或计划使用）游戏服务器主机，那么主机提供商可能已经提供给你可以安装 Sponge 的控制面板了。

注解

当使用 Mojang 的安装程序时，Mojang 使用的是它们自己的 Java 版本而不是你的服务器提供的 Java 版本。安装程序已经内置了 1.8.0_74 版本的 Java。请注意 Sponge 需要不低于 1.8.0_20 的 Java 版本才可以使用；尽管如此，我们仍然推荐使用最新的 Java 8 构建。

SpongeVanilla 只能作为独立服务器运行。

  1. 从Sponge网站下载SpongeVanilla.jar。
  2. 执行命令： java -jar spongevanilla-whatever.jar
  3. 设置 端口转发 以确保其他人能连接至服务器。

警告

不要双击 .jar 文件，否则它会在不显示控制窗口的情况下运行服务器！
