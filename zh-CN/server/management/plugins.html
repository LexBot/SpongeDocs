安装插件¶

插件是什么¶

插件是使用 Java 编写的改变/改善服务器功能的文件。插件允许服主对游戏做出许多修改，如添加经济系统，管理传送和权限等。

寻找插件¶

警告

不要从不可信的地方获取插件！恶意插件可能会导致你的服务器或者计算机被入侵。

SpongePowered 目前运行 Ore 平台 以方便插件开发人员和用户发布和下载插件。您也可以在 SpongePowered 论坛 搜索插件。

安装¶

SpongeForge¶

为了安装插件，需将插件放到您游戏或服务器的 /mods/ 文件夹。如果您下载到的文件是一个 .zip 格式的文件，那么你可能需要解压它以获得一个包含在其中的 .jar 文件。

你也可以将插件放在子文件夹 /mods/plugins/ 中。你甚至可以在 global.conf 中通过 plugins-dir 来自定义插件们所在的文件夹。 SpongeForge 会自动在这些文件夹中搜索插件。但请注意，使用了 Mixins 的插件必须直接放在 mods 文件夹里。

SpongeVanilla¶

注解

为了保持 SpongeForge 和 SpongeVanilla 二者风格一致，SpongeVanilla 的插件也应放置在 mods 文件夹中。

为了安装插件，需将插件放到您游戏或服务器的 /mods/ 文件夹。如果您下载到的文件是一个 .zip 格式的文件，那么你可能需要解压它以获得一个包含在其中的 .jar 文件。

常见问题¶

如果一个插件出现了问题，以下是可能的原因：

  • 插件是否与你使用的 Minecraft 版本兼容？尽管 Sponge 尽力保持对旧版本插件的兼容性，但它仍然会遇到不可能保持向后兼容的时候。大多数时候，插件无需更新就可以在 Sponge 的稳定版发布上工作；但还是请注意，当你升级 Sponge 到下一个主版本号时，插件有可能会停止工作，因为主版本号递增意味着含有不向后兼容的破坏性更新。参阅我们的版本规范获得关于这方面的更多信息。
  • 你的 Java 版本是否保持最新？Sponge 基于 Java 8 构建，并且已知在 JRE 1.8.0_20 上会触发一个 bug，请确保你的 Java 版本比这个要新。
  • 插件可能已经过期，是否有新版本的插件？
  • 这个插件依赖特定的 Sponge 实现或者版本么？有的插件绕过了 SpongeAPI，使用了会随版本改变的底层内容。访问你下载到这个插件的网页获得更多信息。
