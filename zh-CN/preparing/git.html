安装 Git¶

Git 是一个开源的版本控制系统，它极大的方便了开发人员间的合作。

和数以千计的开源项目一样，Sponge 将它的代码托管在 GitHub 上。因此，Git 是 Sponge 及 SpongeAPI 开发中至关重要的工具。

开发者可以在 Git 网站上找到海量文档，以及一系列适用于不同操作系统的 GUI 版客户端。

相关下载¶

Windows¶

GitHub Desktop 是一种在 Windows 上简便地安装 Git 的方式，因为 Git 已经被包含其中，作为软件的一部分。

我们建议在安装完成后重启计算机。

Mac¶

在 macOS 上安装 Git 的方法有很多。

最简单的方法是安装 Xcode Command Line Tools。

警告

下列内容不适用于比 Mavericks 更老的系统。如果你正在使用一个这样的老系统，请安装 Github 客户端。

  1. 启动终端。
  2. 执行 xcode-select --install。
  3. 等待安装，也许你可以乘机来块饼干。
  4. 在终端运行 git

此外，你可以安装 GitHub Desktop。Git 在 GitHub Desktop 的安装过程中作为其中的一部分。

我们建议在安装完成后重启计算机。

Linux 和 Unix¶

在 Linux 上安装 Git 的最简单方法是使用你的 Linux 发行版的包管理系统。

注解

你也许需要在下列命令前加上 sudo。

  1. 启动终端。
  2. 如果你使用 Ubuntu 之类的基于 Debian 的系统，执行 apt-get install git。如果是 Fedora，执行 yun install git。

GitHub Desktop 客户端有 Windows 和 Mac 版，但没有 Linux 版。

我们建议在安装完成后重启计算机。

配置¶

你是谁？¶

在开始使用 git 和 git 仓库之前，请确保你的 git 配置已设置了你的身份。打开你的 CLI 并输入：

git config --list

查找 user.name 和 user.email 。如果它们和你在 GitHub 帐户上的用户名和电子邮件不同，请设置它们：

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

小技巧

在设定完成你的 user.name 和 user.email 之前，请不要对 Sponge 仓库做任何提交。
