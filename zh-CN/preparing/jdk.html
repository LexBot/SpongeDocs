安装 JDK¶

Java Development Kit，又名 JDK，是开发 Sponge 插件或 Sponge 项目本身所必须的。

相关下载¶

在安装 JDK 之前，卸载电脑上所有旧版本的 Java。

注解

Sponge 贡献者以及插件作者必须使用 JDK 8，因其不支持旧版本 Java。请注意，某些 Minecraft 服务器仍未迁移到 Java 8 上。若要正常使用 Sponge，你必须更新到 Java 8，旧版本（即 Java 6 或 7）是无法运行 Sponge 的。

甲骨文公司（Oracle）在其网站上提供了免费的 JDK 下载。确保你安装的是 JDK（Java Development Kit）而非 JRE（Java Runtime Environment），二者并不相同。

  • Java Development Kit 8

你还可以从 AdoptOpenJDK 下载到开源的 JDK。

等待安装程序结束后，重启电脑。JDK 环境就配置好了。
