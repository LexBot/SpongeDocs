Sponge’a Katkı Sağlama¶

Bu bölüm Sponge’a nasıl üye olunacağını gösterir.

Yardımcı olabileceğim projeler nelerdir?¶

Sponge projeleri ve onun parçaları Github üzerinde korunur. Eğer GitHub git aracı ahkkında bilgin yoksa, o zaman önce :doc: howtogit`e bir göz at. Herhangi bir işleme başlamadan önce :doc: `guidelines sayfasını okumanızı tavsiye ederiz. Şuan sürdürdüğümüz çeşitli projeler bulunmaktadır:

  • SpongeAPI
  • SpongeForge
  • SpongeVanilla
  • SpongeCommon
  • Mixin
  • SpongeDocs
  • Cevher

If you want to know more about the structure of the project and how everything is tied together, head over to Sponge Projesinin Yapısı.

Nasıl bir yardıma ihtiyaç var?¶

Temel katkılar¶

Neredeyse herkes tarafında kullanılabilir. Kullanmak için Java yada Python gibi yazılım dili bilmene gerek yok:

  • SpongeForge yada SpongeVanilla’yı test et, program ve kullanım hatalarını bildir
  • karşılaştığın herhangi yanlışları, kusurları, hataları bildir veya öner
  • sponge’u daha iyi hale getirebilecek fikir ve önerileri bildir

Reporting bugs best via our GitHub repositories, suggestions fit onto our forums. Just have a look at our Bug Reporting page for further instructions.

Orta Derece Katkılar¶

Takip edilen görevlerde yardımcı olman için Jawa, Python ve reST uygulamalarında en azından temel bilgisine sahip olman gerekir:

  • hataları düzeltmede yardımcı ol
  • API (SpongeForge ve SpongeVanilla) uyarlamasını tamamla
  • SpongeDocs’un yazımına yardım et
  • Docs’un çevirisinin Crowdin üzerinde yapılmasına yardım et
  • Ore’yi geliştirmede yardımcı ol

The development of the SpongeAPI, SpongeForge, SpongeVanilla and Ore as well as the SpongeDocs is done in several repositories hosted on GitHub.

Gelişmiş Katkılar¶

And finally, these are the most difficult things you can help out with. Advanced knowledge of Java, Minecraft and at least basic knowledge of the SpongeAPI and its structure is strongly advised before attempting to help out with:

  • işlev API’ye ekleniyor (Çekme isteği gönderme)
  • uygulamalarda gelişmiş API işlevselliği uygulamak
