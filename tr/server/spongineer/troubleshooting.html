Sorun Giderme¶

Buradasınız çünkü, Sponge sunucunuz ile birşeyler ters gitti. Hadi ne olduğunu anlamaya çalışıp, ne yapabileceğimize bakalım.

Potential Sources of Trouble

  • Java Bilgisayarınızda Kurulmamış
  • Ağ Bağlantı Başarısızlığı (veya DDoS Atak)
  • Yeterli boş hafıza yok
  • Malformed Config File (e.g. Bad Editing)
  • Eklenti (veya mod) arızalı
  • Operating System Unstable (e.g. Virus Infection)
  • Bozuk Veri
  • Klavye ve sandalya arasında problem
  • Sponge’da bir Bug var
  • Evrende birşeyler ters gidiyor

Java Bilgisayarınızda Kurulmamış¶

Solution: Get Java. Visit the Java kurulumu for more information.

Ağ Bağlantı Başarısızlığı (veya DDoS Atak)¶

Semptomlar: Ağ bağlantısı çok gecikmeli, bağlantı düşüleri sürekli oluyor veya yok.

Çözümler: Bağlantınızı modem veya yönlendirici ile kontrol edin. Tarayıcınız aynı sorunu yaşıyor mu diye kontrol edin. Bağlantınızı test etmek için speedtest.net gibi ücretsiz servisleri kullanabilirsiniz. Bilgisayarınızda çalışan başka servisler probleme sebep olabilir. Yönlendiricinizde bağlantı noktası yönlendirmenizin açık olup olmadığını kontrol edin. Olası DDoS saldırısı bağlantınızı mahvetmiş olabilir. böyle bir durum olduğunu düşünüyorsanız servis sağlayıcınız ile iletişime geçmeniz gerekir.

Yeterli boş hafıza yok¶

Semptomlar: Sunucu çökmesi, Sıkça “Bellek yetersiz” uyarısı alarak.

Solutions: Expand the maximum Perm memory size with the startup argument -XX:MaxPermSize=128. Expand your server heap memory (if possible) with startup arguments e.g. -Xms1024M (1GB starting memory) and -Xmx2048M (2GB maximum). Monitor your free memory on the computer and see if there is some locked up in other processes. You may need to kill frozen java processes, or restart your machine. Memory leaks sometimes occur with bugs in plugins, which can take time to isolate.

Hala sorun var mı?: Hala bir sorun yaşıyor ve yığın boyutunu gelişletemediyseniz, görev yöneticisini açarak tüm belleği kullanabiliyor olduğunuza emin olun. Eğer hala bir sürü boşta bekleyen bellek var ise, 32-bit Java çalıştırıyorsunuz demektir. Eğer 32-bit Java kullanıyorsanız, 64-bit Java’ya işletim sisteminizi de aynı olacak şekilde yükseltmenizi öneririz.

Malformed Config File (e.g. Bad Editing)¶

Semptom: Bir(veya daha fazla) eklenti açılamadı veya beklenmedik bir davranış sergilediyse, başlangıç esnasında sunucu dosyaları okunamaz başlangıç dosyaları içerektir. Sunucu çökebilir ve veri bozulabilir.

Çözüm: Sunucuyu durdurun ve düzenlediğiniz dosyaları kontrol edin. Bozulmuş dosyaların yedeklerini geri yükleyin. Yapılandırma dosyasını komple silebilir ve sunucunun başlangıçta yeniden yüklemesine izin verebilirsiniz.

Eklenti (veya mod) arızalı¶

Semptom: Buna neredeyse herşey sebep olabilir. Eklentiniz ne yaparsa, ek olarak X-factor de yapabilir. Genel olarak zincirleme sunucu çökmeleri, sunucunun kayıt dosyalarında bulunur.

Solution: Stop the server, and check to see nothing has been corrupted. Be sure to check that it isn’t from an incorrectly edited config file (above). Remove suspect plugins and add them again one by one, restarting the server each time. The problem may originate from one plugin that is out of date - check for updates. Plugin conflict may also be the cause, having two incompatible plugins. You can also check our Mod Compatibility page to see if the problem is already known, and if there’s a way to fix it.

Operating System Unstable (e.g. Virus Infection)¶

Semptom: Sunucu devamlı olarak çöküyor, zaman aşımına uğrıyor ve diğer işletim sistemi kısımları da problemli.

Solution: Stop everything. Thoroughly check your system and storage devices for malware and viruses. Good tools for this include AdwCleaner, Junkware Removal Tool, MalwareBytes, and most antivirus ware. Check your server files for corruption after a clean restart of your system. Examine the hardware for damage too if the problems persist - e.g. a faulty power supply.

Bozuk Veri¶

Semptom: Dünya dosyalarında yükleme sorunu oldu veya oyuncular sunucuya girdiğinde sunucu çöktü. Veritabanı bozuk.

** Çözüm **: Yedek dosyalarınızı bozuk veri yüklemelisin. Hasar görmüş dünyalar onarımı için yazılımlar mevcut ve eksik bölgeler yeniden oluşturulabilmektedir. Problemin nedenini araştırın - hatalı biçimlendirilmiş bir eklenti, veritabanı sürücüsü için, elektrik kesilmesi veya başka bir sorunmuydu? Önemli verilerin düzenli yedeklerini güvenli bir aygıta yedeklediğinizden her zaman emin olun.

Klavye ve sandalya arasında problem¶

** Belirti **: Dün her şey düzgün çalışmaktaydı. XYZ’yi yaptıktan sonra bugün sorunlu gibi görünüyor …

** Çözüm **: SpongeDocs, yazılımın öngörülemeyen şekilde başarısız olmasına neden olacak insanların herşeyi kapsamayacak kadar büyük değil. Sunucunuzun sorunsuz çalışmasını etkileyebilecek son zamanlarda yapmış olabileceğiniz şeyler hakkında uzun düşünmek her zaman gereklidir. Bu bellek kartı tozundan sonra problem olmuş olabilir,…

Sponge’da bir Bug var¶

** Belirti **: Yukarıdakilerin hiçbiri geçerli değildir ve halen olması gerektiği gibi çalışmamaktadır.

Çözüm: Büyük silahlardan kurtulma zamanı. SpongeForge veya SpongeVanilla sorun takipçisi üzerinde bir rapor hazırlayın, kullandığınız Forge ve Sponge sürümünün detaylarını ve ilgili sunucu günlük dosyasına bir bağlantı eklemeyi aklınızda tutun.

Evrende birşeyler ters gidiyor¶

Size bununla yardım edemeyiz. Kendi başınızasınız.
