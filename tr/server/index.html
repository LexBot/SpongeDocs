Bir sunucu oluşturma¶

The articles in this section explain how to create and run a server powered by an implementation of SpongeAPI.

Sünger uygulamasıyla yeni bir sunucu oluşturmak isteyen bir kullanıcı iseniz veya varolan bir sunucuyu Sünger uygulama alanına geçirirseniz Eklenti geliştiricileri, bu bölümdeki makalelerle de ilgilenebilmektedir. - test sunucuları, eklenti geliştirme için yararlı olabilmektedir.

Not

We attempt to be implementation-agnostic when possible, meaning that instructions generally aren’t specific to one implementation of SpongeAPI. Nonetheless, mistakes can happen.
