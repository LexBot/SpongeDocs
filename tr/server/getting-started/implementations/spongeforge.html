SpongeForge’u kurmak¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it’s more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Minecraft Forge kullanmak isteyen kullanıcılar :doc:`SpongeVanilla <spongevanilla>.

İndirme¶

Review our Versioning and grab your copy of SpongeForge here.

SpongeForge’u kurmak¶

Not

Eğer oyun sunucusu kullanıyorsanız (veya kullanmayı planlıyorsanız), Sponge’u sizin yerinize kurabilecekleri kontrol paneline sahip olabilir.

Not

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_74. Note that Sponge requires at least 1.8.0_20 or above to run properly but it is recommended to use the latest Java 8 version.

Tekli oyuncu / Oyun-içi local sunucular¶

  1. Minecraft Forge yükleyicisini Minecraft Forge website. adresinden indirin. Yukarı da gösterildiği gibi, Tam tamına aynı yapı numarasını kullandığınıza emin olun.
  2. Sağlanmış olan Forge yükleyiciyi çalıştırın. Yeni bir Forge profili Minecraft başlatıcısında oluşturulacaktır.
  3. Minecraft başlatıcısını açın, yeni bir Forge profili seçin.
  4. “Seçenekler” tıklayın ve “Oyun Dizini Aç” tıklayın.
  5. Sponge web sitesinden SpongeForge indirin ve mods klasörüne atın. Eğer yoksa klasörü oluşturun.
  6. Eğer sunucunuzu online’a açarsanız, pponge hem lokal hem de online olarak çalışabilir.

Sonra şunu öğrenebilirsiniz Sponge yapılandırma ve Sponge örneğini yönet (eklenti kurmak da dahil).

Adanmış sunucular¶

Not

Zaten bir Forge sunucusuna sahipseniz, Sponge modunu mods klasörüne atın. Forge sürümünüzü SpongeForge’nin istediği biçimde güncellemeyi unutmayın. İhtiyacınız olan sürümden emin değilseniz sayfanın üst kısmına bakınız.

Komut istemiyle Forge’u yüklemek¶

  1. ‘Minecraft Forge web sitesini <https://files.minecraftforge.net/>’_ ziyaret edin ve mevcut seçeneklerin tümünü görüntülemek için “Tüm indirmeleri göster”e tıklayın. Listelenenl eşleşen sürümü tanımlayın :doc: ‘SpongeForge’un dosya adında <spongeforge>’ u indirin ve direk indirme linkine geçmek için (i) yi “Yükleyici”nin üzerine getirin.
  2. Jar’ı hedefine indirmek için favori indirme yönteminizi kullanın. Örnek: wget http://url.to/forge-version-installer.jar
  3. Forge yüklemek istediğiniz klasörden, jarı --installServer seçeneğiyle çalıştırın. Örnek: java -jar forge-version-installer.jar --installServer
  4. Forge için SpongeForge aşağıya eklemeye devam edin.

GUI üzerinden Forge yükleniyor¶

  1. Minecraft Forge website adresinden MinecraftForge yükleyicisini indirin. SpongeForge dosya ismiyle eşleşen bir sürümü versiyon uyumluluğu için seçin. Yukarıya bakarak, Spongeforge ve Forge’un isimlendirme şemasını görebilirsiniz.
  2. Sağlanan Forge yükleyicisini çalıştırın, “Install Server“‘ı seçin, sunucunun dosyalarını yerleştirmek için boş bir klasör seçin ve TAMAM’ı tıklayın.
  3. Forge için SpongeForge aşağıya eklemeye devam edin.

SpongeForge’u Forge’a eklemek¶

  1. Sponge web sitesinden SpongeForge’u indirin ve sunucu dizininizdeki “mods” klasörüne koyun. Henüz yoksa klasörü oluşturun.
  2. Artık sunucuyu java -jar forge-version-XYZ.jar komutuyla başlatabilir veya komut istemiyle başlatabilirsiniz.
  3. Evden çalıştırılıyorsa, başkalarının bağlanabilmesi için Port Yönlendirme ı ayarlayın.

Sonra, :doc: script komutu <../launch-script>, :doc: Sponge yapılandır <../configuration/index> ve :doc: sunucunuzu yönetin </server/management/index> ı nasıl oluşturacağınızı ve kullanacağınızı öğrenin`(eklentileri kurma da dahil ).
