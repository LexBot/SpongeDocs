Sponge Sözlüğü¶

This is a guide to the more common reserved words, phrases and abbreviations used in the Sponge Project. Links are provided in some cases to provide more information. It is not intended to be a guide to SpongeAPI or Java.

Not

Şartlar minecraft ve/veya Sponge projesine istinaden tanımlanır.

Tanımlar¶

API
Uygulama programlama arayüzü- mod yapmanın kolay yolu.
Katkıda bulunan
Spongepowered projesine katkıda bulunmuş kimseler.
Coremod
CoreMod, erişim transformatörleri, ASM sınıf değişikliği veya diğer Java bytecode manipülasyonu yoluyla yerel Minecraft kodunu değiştirmesine olanak tanıyan Mojang sınıfları yüklenmeden önce yüklenen Forge modudur.
Crowdin
Dökümantasyonlar için web tabanlı çeviri sistemi. SpongeDocs birçok dilde belgeler sağlamak için kullanır. https://crowdin.com/project/sponge-docs
Geliştirici
Kodu yazanlardan biri ( Sponge, plugins, Mods, …).
Discord
Discord is a free voice, video and text chat app, and is used as a chat platform. Like IRC, Provides chat rooms for all things Sponge. Sponge’s Discord server is synced with Sponge’s IRC server.
FLARD
Yeşil, yenilebilir ve patlayıcı. Onu rahat bırak ki oda seni rahat bıraksın.
Forge
Uzun dönemli minecraft modding API, kendi yükleyicisi ile. Forge tek kişilik oyunda, yerel ağda ve sunucularda ki eklentileri aktif etmenizi sağlar. https://mcforge.readthedocs.io/en/latest/
GitHub
Https://github.com/ web tabanlı sistemleri, kodları paylaşmak ve proje oluşturmak için ücretsiz sistem
Uygulama
A mod or program that enables the use of SpongeAPI in Minecraft.
IRC(internet sohbeti)
Internet Relay Chat, used as a chat platform. Provides chat rooms for all things Sponge. Sponge’s IRC server is synced with Sponge’s Discord server.
Sorun
Sorunları, hataları, iyileştirmeleri ve GitHub’da ve başka yerlerde yapılacak işleri işaretlemenin bir yolu. Sponge projesi, GitHub’ı bir sorun izleyici olarak kullanıyor.
Java
Minecraft ve sponge’un yazıldığı programlama dili, çalışması için gerekir. https://java.com/en/
MCP
Eklenti kodlayıcı paketi, geliştiricilerin hem Minecraft sunucusu hem de istemci için mod oluşturmasına yardımcı olur. http://www.modcoderpack.com/website
Mixins
Specialized program components that inject Sponge into Minecraft. https://github.com/SpongePowered/Mixin/wiki
Eklenti
Oyunu bir şekilde değiştiren Minecraft değişikliği. Forge API’sini kullanarak yazılmış modlar Forge’a ihtiyaç duyarlar, ancak bazı modlar kendi başına yüklenebilir.
Oyuncu olmayan karakter
Non-Player Character. Any character not controlled by a player, e.g. a Villager.
Cevher
The Official Sponge plugin hosting site. https://ore.spongepowered.org/
Patreon
Projelere ve organizasyonlara bağış sağlayan bir hizmettir. Sponge’u desteklemek istiyorsanız, bakmanız gereken yer burasıdır! Sünger Patreon Bağış Sayfası <https://www.patreon.com/Sponge> __
Eklenti
A Minecraft mod that extends and changes Minecraft via SpongeAPI, usable only on Sponge servers.
Proje liderleri
The folks in charge of the entire Sponge Project, responsible for coordinating the activities of the various Teams and sub-projects.
Çekme isteği (PR)
Açık kaynak bir projeye katkıda bulunma metodu, github üzerinde ki sponge deposu gibi.
Sponge
Tüm projenin adı.
SpongeAPI
Sponge projesi tarafından geliştirilen Minecraft Plugin API’sı.
SpongeAuth
The authentication panel and SSO solution for all of the Sponge websites. https://auth.spongepowered.org/
SpongeCommon
Resmi SpongeAPI uygulamalarında ortak olan ve bunları oluşturmada kullanılan bir kod deposu. SpongeVanilla ve SpongeForge bu depoyu kullanıyor.
SpongeForge
Bir Forge Coremod olarak SpongeAPI uygulanması.
SpongeDocs
Sünger hakkındaki en iyi bilgi kaynağı olan ve sıkça güncellenen belgeler.
Sponge forumları
Home to the Sponge Project and the second-best source of information on Sponge. Update announcements are posted regularly. https://forums.spongepowered.org/
Sponge vakfı
Projenin mali tarafı, bağış kabul eden, yöneten ve dağıtan ayrı bir varlıktır.
SpongeVanilla
An implementation of SpongeAPI that runs on Vanilla Minecraft (and not Forge).
Spongie
Sponge Maskot, kırmızı bir pelerinle cesur küçük bir sünger.
Spongy
Dokümanlar için PR önizlemelerini yapan Sponge GitHub Botu.
Takım liderleri
The people who direct the Teams (WebDev, Docs, SysOps, subreddit, IRC).
