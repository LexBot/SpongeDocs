Instalando Git¶

Git es un sistema de control de versiones de código abierto, y ayuda tremendamente con el desarrollo de proyectos colaborativos.

The Sponge project, alongside thousands of other open-source projects, hosts its Git repositories on GitHub. Thus, Git is a crucial tool in the development of Sponge and SpongeAPI.

La pagina web Git tiene una sustancial documentación, y su página de descargas ofrece una gama de opciones para clientes de GUI aptos para varios sistemas operativos.

Descargar¶

Windows¶

GitHub Escritorio es un método fácil de instalar Git en Windows, porque Git está incluido como parte del software.

Se recomienda reiniciar su computadora después de instalar Git.

Mac¶

Hay diversas maneras de instalar Git en macOS.

El método mas fácil es instalar las Herramientas de Línea de Comandos Xcode.

Advertencia

Estas instrucciones no funcionan en un Macs que ejecuten una versión anterior a Mavericks. Si usted está usando una versión de OS X anterior a Mavericks, instale el cliente de GitHub en su lugar.

  1. Lanzamiento del Terminal.
  2. Ejecute xcode-select --install.
  3. Instálalo, y agarra una galleta mientras esperas.
  4. Ejecute git desde la Terminal.

Alternativamente, Ud. puede instalar GitHub Escritorio. Git esta disponible como una parte de la instalación de escritorio de GitHub.

Se recomienda reiniciar su computadora después de instalar Git.

Linux y Unix¶

El método más sencillo para instalar Git en Linux es utilizando el gestor de paquetes que vino con tu distribución de Linux.

Nota

Puede que necesite prefijar esos comandos con “sudo”.

  1. Lanzamiento del Terminal.
  2. Run apt-get install git if you are on an Ubuntu or Debian-based distribution. Run yum install git if you are on Fedora.

GitHub Escritorio no está disponible actualmente para Linux, a diferencia de Windows y Mac.

Se recomienda reiniciar su computadora después de instalar Git.

Configuración¶

¿Quién Eres?¶

Antes de que Ud. siquiera empiece a trabajar con git y el repositorio, asegúrese de que su configuración de git tenga su identidad establecida. Abra su CLI e ingrese:

git config --list

Busque user.name y user.email. Si ellos no tienen el mismo nombre de usuario y correo que su cuenta de GitHub configúrelos:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Truco

No proceda con ninguna contribución en ningún repositorio Sponge hasta que Ud. establezca su user.name y user.email.
