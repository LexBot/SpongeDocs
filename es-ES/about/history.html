La Historia de Sponge¶

Esta página está dividida en dos partes principales:
  • la historia del proyecto
  • la historia de nuestra mascota Spongie

La Historia del Proyecto¶

Sponge was founded as a better alternative to the APIs that were available as of September 2014. The Sponge Equipo consists of many people from different Communities, e.g. Spout/Spoutcraft, Forge, Cauldron and a few others.

Cuando el desarrollo de Bukkit y caldero llegó a un abrupto final, la comunidad de Minecraft se sorprendió. Varios desarrolladores de las mencionadas comunidades se reunieron en #nextstep en Esper.NET y discuten el futuro de Minecraft modding. Se decidió que ciertamente había una demanda de una nueva API como repositorio de CraftBukkit (caldero también) fue tomado.

Algunos objetivos que deben lograrse con la nueva API:
  • consistencia
  • estabilidad a través de actualizaciones de Minecraft
  • fácil de usar
  • conformidad con el Acuerdo de Licencia de Mojang
  • protección contra caidas DMCA

While the goals were mostly clear, the route to be taken was not. The soon-to-be Project was still nameless too. The first mention of Sponge as the projects name was on Sept, 6th by Firehead94. Obviously, the name stuck.

The initial commit to SpongeAPI and SpongeForge was made on September 7th and 8th, 2014 by one of the Sponge Project leaders, Zidane. This was the beginning of the development of SpongeAPI and SpongeForge (named Sponge at the time).

On September 7th, 2014 the initial commit to Granite, an API based upon Vanilla Minecraft was made. Granite, originally started as an independent project, was planned to implement its own API and SpongeAPI. Granite and SpongeForge coexisted until April 20th, 2015.

The development of SpongeForge and the API gained momentum leading to a first API release on December 1st, 2014. However, this version was far from feature complete and an API-only release, meaning that there was no official implementation available at that time.

En 26 de diciembre del 2014, el equipo de granito decidió limitar el granito para el uso de SpongeAPI. Eso hizo granito la primera aplicación no oficial de esponja de vainilla Minecraft. El 26 de marzo, el equipo de granito finalmente se unió al equipo de esponja y granito se convirtió en SpongeVanilla, la aplicación de SpongeAPI oficial de Minecraft de vainilla. Un mes más tarde la unificación de la base de granito y SpongeForge desde SpongeCommon. SpongeCommon fue creado para contener código todo compartido entre las dos implementaciones aunque específicos de la implementación código permanece en SpongeVanilla y SpongeForge.

El 21 de abril de 2015 la API Sponge 2.0 fue lanzada finalmente. Desarrollo de 25 de abril de 2015 en estructuras de SpongeForge (#428, inspirado Wallaroo) finalmente fueron puestos a disposición. La primera versión alfa de SpongeVanilla (DEV #17) fue lanzada el 13 de septiembre de 2015.

En 31 de diciembre del 2015 la Api Sponge 3.0.0 fue lanzada, junto con SpongeForge (#1000) y SpongeVanilla (#129) en versión beta.

Spongie - nuestra mascota¶

Spongie es nuestra mascota amigable, servicial y algo empapada. No la hagas llorar, ya que ella es una persona muy sensible.

La Historia de Spongie¶

Spongie first appeared in #Sponge on Esper.NET around September 2014. She was created and posted by DragonsPainter. Strad, another user, felt that Spongie would look better with a moogle-like antenna, a Forge furnace instead of a Cauldron and replaced the Spigot with Spout. An anonymous user then removed all labels and introduced her as the icon and background logo for the Sponge reddit section.

Esta versión de Spongie lamentablemente no está disponible como gráfico vectorial. Así que el personal de sponge decidió crear una nueva versión vectorizada. Dos nuevos proyectos fueron luego publicados en “Estado de Spongie <https://forums.spongepowered.org/t/state-of-spongie-wip/6194>`__.
