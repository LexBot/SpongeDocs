Instalando SpongeForge¶

SpongeForge integrates Minecraft Forge so you can also run Minecraft Forge mods. In fact, it’s more like Sponge itself is a Forge mod that then loads Sponge plugins, but this is a technical detail.

Los usuarios que no quieran usar Minecraft Forge pueden considerar SpongeVanilla.

Descargar¶

Review our Versioning and grab your copy of SpongeForge here.

Instalando SpongeForge¶

Nota

Si usa (o planea usar) un hospedador de servidor de juegos, es posible que ellos tengan un panel de control que pueda instalar Sponge por usted.

Nota

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_74. Note that Sponge requires at least 1.8.0_20 or above to run properly but it is recommended to use the latest Java 8 version.

Un Solo Jugador / Servidores LAN En-El-Juego¶

  1. Descargue el instalador de Minecraft Forge desde el sitio web Minecraft Forge. Asegúrese de usar exactamente el mismo número de compilación como se muestra arriba.
  2. Ejecute el instalador de Forge proporcionado. Se creará un nuevo perfil de Forge en el iniciador de Minecraft.
  3. Abra el iniciador de Minecraft y seleccione el nuevo perfil de Forge.
  4. Haga clic en «Options» y haga clic en «Open Game Dir».
  5. Descargue SpongeForge del sitio web de Sponge y colóquelo en la carpeta mods. Cree la carpeta si aún no existe.
  6. Sponge debería funcionar tanto en un solo jugador como si abres tu mundo a un LAN.

Luego, aprenda cómo puede configurar Sponge y cómo gestionar su instancia de Sponge (incluyendo la instalación de complementos).

Servidores dedicados¶

Nota

Si ya tiene un servidor Forge, simplemente coloque el mod de Sponge en su carpeta mods. Recuerde actualizar su versión de Forge para que coincida con la que requiere SpongeForge. Eche un vistazo a la parte superior de esta página si no está seguro de la versión que necesita.

Instalando Forge a través de Commandline¶

  1. Visite el sitio web de Minecraft Forge y haga clic en «Show all downloads» para ver el conjunto completo de opciones disponibles. Identifique la versión que coincida con la que figura en la lista en el nombre de archivo de la descarga de SpongeForge, y desplace el cursor sobre el (i) junto a «Installer» para obtener el enlace de descarga directa.
  2. Use su método de descarga favorito para descargar el jar a su destino. Ejemplo: wget http://url.to/forge-version-installer.jar
  3. Desde la carpeta en la que desea instalar Forge, ejecute el jar con la opción --installServer. Ejemplo: java -jar forge-version-installer.jar --installServer
  4. Continúe a Agregando SpongeForge a Forge a continuación.

Instalando Forge a través de GUI¶

  1. Descargue el instalador de Minecraft Forge del sitio web de Minecraft Forge para la versión que coincida con la listada en el nombre de archivo de la descarga de SpongeForge. Ver arriba para el esquema de nombres de SpongeForge y Forge.
  2. Ejecute el instalador de Forge proporcionado, seleccione «Install Server», elija una carpeta vacía para colocar los archivos del servidor y luego haga clic en OK.
  3. Continúe a Agregando SpongeForge a Forge a continuación.

Agregando SpongeForge a Forge¶

  1. Descargue SpongeForge del sitio web de Sponge y colóquelo en la carpeta mods en su directorio de servidor. Crea la carpeta si aún no existe.
  2. Ahora puede iniciar el servidor mediante el comando o ejecutar lo siguiente java -jar forge-version-XYZ.jar.
  3. Si está operando desde su casa, configure Puerto de Reenvio para asegurarse de que otros puedan conectarse.

Luego, aprenda cómo puede crear y usar un launch-script, configurar Sponge y :doc: `administrar su servidor </server/management/index>`(incluyendo la instalación de complementos).
